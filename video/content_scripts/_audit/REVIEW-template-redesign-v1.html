Definition

One-to-One

A function is one-to-one when different inputs always give different outputs.

$f(x_1)\ne f(x_2)\ \text{whenever}\ x_1\ne x_2$
$f(x_1)=f(x_2)\implies x_1=x_2$

Why it matters

Being one-to-one is exactly what lets us run $f$ backwards — no two inputs collide, so every output has one source.

theorem_proof — accent 換成 NTU 金

Theorem

An Inverse Exists Exactly When One-to-One

A function has an inverse if and only if it is one-to-one.

Proof

If $f$ has an inverse and $f(x_1)=f(x_2)$, apply $f^{-1}$: $x_1=x_2$. So $f$ is one-to-one.
If $f$ is one-to-one, each output $y$ has exactly one source $x$ — send $y$ back to it. That is $f^{-1}$.
So one-to-one and invertible name the same thing.
§1.1 · Inverse Functions●  Theorem · 08
