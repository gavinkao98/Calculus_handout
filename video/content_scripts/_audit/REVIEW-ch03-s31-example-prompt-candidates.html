A Companion Limit
$\dfrac{1-\cos\theta}{\theta} = \dfrac{(1-\cos\theta)(1+\cos\theta)}{\theta\,(1+\cos\theta)} = \dfrac{1-\cos^{2}\theta}{\theta\,(1+\cos\theta)}$multiply by $1{+}\cos\theta$
$= \dfrac{\sin^{2}\theta}{\theta\,(1+\cos\theta)} = \dfrac{\sin\theta}{\theta}\cdot\dfrac{\sin\theta}{1+\cos\theta}$$1{-}\cos^{2}\theta=\sin^{2}\theta$
$\longrightarrow\ 1\cdot\dfrac{0}{1+1} = 0$the limit, then continuity

All Six Derivatives
$\dfrac{d}{dx}\tan x = \dfrac{(\cos x)(\cos x)-(\sin x)(-\sin x)}{\cos^{2} x}$quotient rule
$= \dfrac{\cos^{2} x+\sin^{2} x}{\cos^{2} x} = \dfrac{1}{\cos^{2} x} = \sec^{2} x$Pythagorean identity
$\dfrac{d}{dx}\sec x = \sec x\tan x$secant, same rule
$\dfrac{d}{dx}\cot x = -\csc^{2} x,\quad \dfrac{d}{dx}\csc x = -\csc x\cot x$the other two

$\dfrac{d}{dx}\sec x = \sec x\tan x$secant, same rule
