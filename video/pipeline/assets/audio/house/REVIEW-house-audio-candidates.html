House Audio Candidates

這份是課程影片音樂／音效候選裁決稿。全部音檔都由 repo 內的 Python 腳本程序合成，沒有第三方 sample、素材庫音檔或外部 API。

我的初選建議： 先選 Candidate B 當全章節預設。它比 A 更低調，適合長時間數學教學；如果你想要影片幾乎保持自然講課感，選 C；如果你想要比較明顯的品牌開場感，選 A。

裁決方式

裁決項目 要選什麼 我的建議
整套 house style A / B / C 擇一。原則上整套選，不要每個 cue 混搭，影片才會一致。 B
是否使用 caution ping 可選「使用」或「不用」。若覺得教學太像 UI 提示音，就不用。 只在 notation trap 使用
是否保留 divider stinger 可選「使用」或「不用」。若你希望完全自然講課，可以只留 intro/outro。 使用，但音量要淡

套用規則

模板 聲音規則
intro 使用裁決出的 intro bed，只在開場 6-8 秒使用，不延伸進第一個 content scene。
divider 使用裁決出的 divider stinger，僅標示 act/stage 切換。
outro 使用裁決出的 outro bed，只在 end slate 使用。
callout type: caution 可使用裁決出的 caution ping；若覺得太像 UI，可裁掉。
所有推導／定義／證明／表格／符號圖 不放 BGM，不放逐步 reveal 音效，維持旁白清楚。

授權狀態

本候選包目前沒有使用 YouTube Audio Library、Pixabay、Freesound 或其他第三方音檔。所有候選都由 generate_house_cues.py 合成，因此沒有署名需求、Content ID 來源風險或素材庫授權變動問題。
