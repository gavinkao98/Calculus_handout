②a §1.1 Example 1.7：反轉有理函數 + 雙向驗證 建議主錨 78 字

handout/fragments/ch01/sec-1-1.html:274–280

Let \(y = \frac{3}{x - 5}\). Solving for \(x\): \[ x - 5 = \frac{3}{y}, \quad\text{so}\quad x = 5 + \frac{3}{y}. \] Interchanging the variables, \[ f^{-1}(x) = 5 + \frac{3}{x} = \frac{5x + 3}{x}, \qquad x \neq 0. \] Verification, in both directions: \[ \begin{aligned} f\!\bigl(f^{-1}(x)\bigr) &= \frac{3}{\bigl(5 + \tfrac{3}{x}\bigr) - 5} = \frac{3}{3/x} = x, \\ f^{-1}\!\bigl(f(x)\bigr) &= 5 + \frac{3}{3/(x - 5)} = 5 + (x - 5) = x. \end{aligned} \] Both compositions return the input, confirming Proposition 1.1. Note how the domains pair up: \(f\) excludes \(x = 5\) and \(f^{-1}\) excludes \(x = 0\), matching the fact that \(f\) never takes the value \(0\) and \(f^{-1}\) never takes the value \(5\).

為何代表語聲： 全書解法散文最佳標靶——動作標籤式 connective（Solving for／Interchanging／so）、自願加做的雙向 verification（自學者該被示範的習慣）、結尾把具體算式收束回 Proposition 1.1，並把 x≠5／x≠0 翻成「f 不取 0、f⁻¹ 不取 5」的對偶意義。不偷步、每個等號可追、回到「為什麼」。

burstiness/具體： 短指令句（「Interchanging the variables,」3 字）對比結尾 40 字長解釋句，模擬動手短、回望長的節奏；具體算式 100% 在抽象命名之前。

②b §1.5 Example 1.30：0/0 形式有理化分子 配套短解法 27 字

handout/fragments/ch01/sec-1-5.html:158–161

Direct substitution gives \(\tfrac{0}{0}\), so we rationalize the numerator: \[ \begin{aligned} \frac{\sqrt{t^{2} + 9} - 3}{t^{2}} \cdot \frac{\sqrt{t^{2} + 9} + 3}{\sqrt{t^{2} + 9} + 3} &= \frac{t^{2} + 9 - 9}{t^{2}\bigl(\sqrt{t^{2} + 9} + 3\bigr)}\\ &= \frac{1}{\sqrt{t^{2} + 9} + 3}, \qquad t \neq 0. \end{aligned} \] Therefore \[ \lim_{t \to 0} \frac{\sqrt{t^{2} + 9} - 3}{t^{2}} = \lim_{t \to 0} \frac{1}{\sqrt{t^{2} + 9} + 3} = \tfrac{1}{6}. \]

為何代表語聲： 「diagnose → act → conclude」最小三段式（Direct substitution gives 0/0 → so we rationalize → Therefore = 1/6），當主錨太長、需要「短解法語聲下限」時的配套標靶。connective 鏈 so…Therefore 乾淨；動機（為何有理化）先於機械操作。

③a §1.6 開場歷史段（Newton/Leibniz → 19 世紀 ε-δ） 建議 96 字

handout/fragments/ch01/sec-1-6.html:12

From the time of Newton and Leibniz, calculus was remarkably successful at solving problems in geometry, mechanics, and astronomy. For a long time, however, there was also controversy about its logical foundations. Early arguments often relied on infinitesimals — quantities thought of as being extremely small but not described in a fully precise way — and mathematicians continued to ask what it really means to say that \(x\) approaches \(a\) or that \(f(x)\) approaches \(L\). The modern resolution, developed in the nineteenth century, is the \(\varepsilon\)-\(\delta\) definition of a limit, which gave calculus the level of rigor used in modern mathematics.

為何代表語聲： 「歷史旁註不 puffery」典範——把歷史當「concept 為何存在」的動機論證，而非名人軼事。無「偉大天才／劃時代突破」之類空泛宏大語；對 infinitesimals 誠實點出問題（extremely small but not described in a fully precise way）。具體 problem domain 先於抽象的 logical foundations。

burstiness/具體： 長開場句（三領域並列）→ 短轉折句 → em-dash 夾註的最長句 → 中長 payoff 句收束；具體錨（geometry/mechanics/astronomy、infinitesimals）先於抽象（rigor）。

③b §1.6 工程容差段（ε、δ 當 tolerances） ＝ ④b 同一段 78 字

handout/fragments/ch01/sec-1-6.html:25

It helps to read \(\varepsilon\) and \(\delta\) as tolerances. In engineering and manufacturing one specifies how far a finished quantity may stray from its target: an output tolerance \(\varepsilon\) around the desired value \(L\). One then asks how tightly the input must be controlled near \(a\) to keep the result within that tolerance — an input tolerance \(\delta\). The definition below makes this everyday promise precise: control the input closely enough, and the output stays within specification.

為何代表語聲： 「應用旁註不空泛宏大」最佳樣本——不喊「微積分無所不在」，而把具體製造業情境直接映射到待定義的數學物件（output tolerance↔ε、input tolerance↔δ）。收尾收束回即將給出的 definition。

①a §1.3 極限動機段（位置/速度 → 一般差商 → 極限） 建議加強（concrete-first） 138 字

handout/fragments/ch01/sec-1-3.html:12–16

A frequent setting in calculus is the need to study a ratio of changes. If \(s(t)\) is the position of a moving object at time \(t\), the ratio \[ \frac{s(t + h) - s(t)}{h} \] is the average velocity of the object from time \(t\) to time \(t + h\). Very often we want the instantaneous velocity — how fast the object is moving at a single moment — rather than the average. To describe this, we let \(h\) shrink toward zero and ask what value the ratio approaches. More generally, for any function \(f\), the ratio \[ \frac{f(y) - f(x)}{y - x} \] measures how \(f\) changes between two inputs. We may want to know what this ratio does as \(y\) gets closer and closer to \(x\), without ever equalling \(x\). This is the motivation for the concept of a limit.

為何代表語聲： 與既有範文（§1.1，abstract-first 從可逆性切入）互補——本段示範 Stewart 招牌的 concrete-to-abstract：先給可觸摸的 position/velocity，再 More generally 推到任意 f 的差商，最後才命名 limit。明確 connective（Very often／To describe this／More generally）；乾淨交棒句「This is the motivation for the concept of a limit.」

burstiness/具體： 11 字破題句 → 跨 display 長條件句 → 短指令句 → 9 字收束句，起伏明顯；物理可感量在前、抽象差商在後。

①b §1.2 反三角動機段（坡度/三角形比 → arctan/arcsin） 次選 89 字

handout/fragments/ch01/sec-1-2.html:13

The trigonometric functions are not one-to-one on all of \(\mathbb{R}\), so they do not have inverse functions on their full domains. Inverse trigonometric functions arise whenever we know a ratio and want to recover an angle: from the slope of a ramp, we get an angle of inclination via \(\arctan\); from a ratio of sides in a right triangle, we get an angle via \(\arcsin\) or \(\arccos\). To build these inverses, we must first restrict the domain of each trigonometric function to an interval on which it is one-to-one.

為何代表語聲： application-driven 動機（從具體斜坡坡度、三角形邊比帶出形式工具），並把本節接回前一節的 one-to-one。比 ①a 短、密度略低，列次選；若型①只取一段，①a 更全面。

④a §1.6 challenge-and-response（ε-δ 當「挑戰—應招」遊戲） 建議 94 字

handout/fragments/ch01/sec-1-6.html:75

It is useful to picture the definition as a challenge-and-response game. A skeptic names a tolerance \(\varepsilon > 0\) — how close to \(L\) they insist \(f(x)\) must be — and our job is to respond with a \(\delta > 0\) that meets the demand: every \(x\) within \(\delta\) of \(a\) (but not equal to \(a\)) is sent to within \(\varepsilon\) of \(L\). The limit equals \(L\) precisely when we can answer every challenge, no matter how small the \(\varepsilon\). This is why \(\varepsilon\) comes first and \(\delta\) second: \(\delta\) is chosen in reply to \(\varepsilon\).

為何代表語聲： 型④最強標靶——把 Def 1.13 的巢狀 quantifier（∀ε ∃δ）整段翻成白話敘事卻完全不 hedge（無 basically／sort of），果斷說出直覺，再用末句「This is why ε comes first and δ second」收緊釘回 quantifier 順序。與既有範文那句貼在 definition body 內的單句 inline gloss 互補（既有＝定義內單句；本段＝定義外延伸 narrative）。

burstiness/具體： 11 字定調句 → 帶雙破折號＋冒號的 45 字長句 → 兩記短促形式收緊句；擬人具象（skeptic/challenge/demand）先於抽象結論。
