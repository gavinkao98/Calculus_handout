Example 2.12　A vertical tangent: \(f(x)=\sqrt[3]{x}\) at \(x=0\)核心

sec-2-3.html，corner 例＋Figure 2.3 之後 ・ [source: CLP-1 §2.2 prob_s2.2 vertical-tangent remark + standard textbook example]（CC BY-NC-SA 4.0）・解材：authored

填 L1-1：Definition 2.3 提「no vertical spike」，但原僅 Example 2.11 示範 corner。本例示範差商 \(1/h^{2/3}\to+\infty\) 的 vertical tangent——左右極限一致但無界，與 corner 的左右不等對比，是不同的失敗模式。

Show that \(f(x) = \sqrt[3]{x} = x^{1/3}\) is not differentiable at \(x = 0\), even though its graph has no corner there.

We form the difference quotient at \(a = 0\):

\[ \frac{f(0 + h) - f(0)}{h} = \frac{h^{1/3} - 0}{h} = \frac{1}{h^{2/3}}. \]

Since \(h^{2/3} > 0\) for all \(h \ne 0\), the quotient is always positive. As \(h \to 0\) the denominator \(h^{2/3} \to 0\), so \(\displaystyle\lim_{h \to 0} \frac{1}{h^{2/3}} = +\infty\). The limit does not exist as a finite number, so by Definition 2.3 the function is not differentiable at \(x = 0\).

The contrast with Example 2.11: there the one-sided limits existed but disagreed (\(+1\) vs \(-1\)); here both tend to \(+\infty\), so there is no corner — instead the graph has a vertical tangent at the origin.

Example 2.13　Piecewise differentiability選收

sec-2-3.html，Theorem 2.1 的 Caution 之後 ・ [source: CLP-1 §2.2 prob_s2.2 "2006H piecewise x^2 / ax+b"]（CC BY-NC-SA 4.0）・解材：official（改寫）

填 L2-1：把 differentiability 定義反過來用——求 \(a,b\) 使 piecewise 處處可微，是 Theorem 2.1 的直接應用（先用連續、再用左右導數相等）。

Find the constants \(a\) and \(b\) for which \(f(x) = \begin{cases} x^{2} & x \le 2, \\ ax + b & x > 2 \end{cases}\) is differentiable everywhere.

Away from \(x = 2\) each piece is a polynomial, so the only point in question is \(x = 2\).

Step 1 — Continuity. By Theorem 2.1, differentiability at \(x = 2\) requires continuity, forcing \(\lim_{x\to2^-}f = 4\) and \(\lim_{x\to2^+}f = 2a+b\) to agree: \(2a + b = 4\).

Step 2 — Matching derivatives. The left derivative uses the \(x^2\) branch, whose derivative \(2x\) follows in one line from the difference quotient (the same technique used in §2.2 Example 2.6), giving slope \(4\). The right derivative is the constant \(a\). For the two-sided limit to exist, \(a = 4\).

Then \(2(4) + b = 4\) gives \(b = -4\). With \(a = 4,\ b = -4\) the pieces meet smoothly and the difference quotient at \(x = 2\) has limit \(4\).

Example 2.16　Power rule, negative exponent核心

sec-2-4.html，Example 2.15 之後、exponential 小節之前 ・ [source: CLP-1 §2.6 prob_s2.6 "s(t)=3t^4+5t^3-1/t"]（CC BY-NC-SA 4.0）・解材：official（改寫）

填 L1-2：Caution 說 power rule 對負整數指數亦成立，但原例只用正整數次方。本例把 \(1/t\) 改寫 \(t^{-1}\) 再套規則，回扣 §2.2 Example 2.8 的 \((1/x)'=-1/x^2\)。

Differentiate \(s(t) = 3t^{4} + 5t^{3} - \dfrac{1}{t}\).

Rewrite \(1/t = t^{-1}\): \(s(t) = 3t^{4} + 5t^{3} - t^{-1}\). Applying the power rule (including the negative-exponent case from the Caution above) term by term,

\[ s'(t) = 3\cdot 4t^{3} + 5\cdot 3t^{2} - (-1)t^{-2} = 12t^{3} + 15t^{2} + \frac{1}{t^{2}}. \]

The \(+1/t^{2}\) term matches what the quotient rule (§2.5) or the limit definition (§2.2 Example 2.8) would give.

稽核 ✓ final answer \(12t^3+15t^2+1/t^2\) 正確、各步有效、交叉引用解析。RESOLVED none

Example 2.17　Power rule, fractional exponent + tangent line核心

sec-2-4.html，Example 2.16 之後 ・ [source: CLP-1 §2.4 prob_s2.4 "f(x)=3x^2+4x^(1/2)"]（CC BY-NC-SA 4.0）・解材：official，tangent-line 部分本書新增

填 L1-2（延續）：補分數指數 \(x^{1/2}\)，並回扣 §2.1 切線技能；驗證與 §2.2 Example 2.7 從定義所得一致。

Differentiate \(f(x) = 3x^{2} + 4\sqrt{x}\) for \(x > 0\), and find the tangent line at \(x = 1\).

Writing \(\sqrt{x} = x^{1/2}\), we differentiate term by term. The power rule applies directly to \(3x^{2}\). For \(4\sqrt{x}\) the exponent \(\tfrac{1}{2}\) is not an integer — the case the Caution above defers for a general proof — but this particular derivative is already secured: §2.2 Example 2.7 found \(\dfrac{d}{dx}\sqrt{x} = \dfrac{1}{2\sqrt{x}}\) straight from the definition. Using it,

\[ f'(x) = 3\cdot 2x + 4\cdot\tfrac{1}{2}x^{-1/2} = 6x + \frac{2}{\sqrt{x}}. \]

At \(x = 1\): \(f(1) = 7\) and \(f'(1) = 8\), so the tangent line is \(y - 7 = 8(x - 1)\), i.e. \(y = 8x - 1\).

Example 2.21　Quotient-rule error spotting核心

sec-2-5.html，Example 2.20（首個 full quotient 例）＋其圖之後 ・ [source: CLP-1 §2.6 prob_s2.6 Conceptual Q1 "spot the error: 2x/(x+1)"]（CC BY-NC-SA 4.0）・解材：official（改寫）

填 L2-3：§2.5 的 Caution 點名「swapping the order flips the sign」是最常見 quotient-rule 錯誤，但無以辨錯為核心的 example。本例給含 sign＋化簡兩錯的學生計算要求糾正。

A student differentiates \(f(x) = \dfrac{2x}{x+1}\) and gets \(\dfrac{2(x+1)+2x}{(x+1)^2} = \dfrac{2}{x+1}\). Spot all the errors and give the correct derivative.

Error 1 — wrong sign. The quotient rule has a minus: \(\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}\); the student wrote a plus. The correct numerator is \((2)(x+1) - (2x)(1) = 2\).

Error 2 — bad simplification. Even the student's own numerator is \(2(x+1)+2x = 4x+2\), not \(2(x+1)\); the \(2x\) term was dropped.

Correctly, \(f'(x) = \dfrac{(2)(x+1) - (2x)(1)}{(x+1)^2} = \dfrac{2}{(x+1)^2}\) — note the denominator is \((x+1)^2\), not \((x+1)\).

稽核 ✓ 兩錯辨識正確、正解 \(2/(x+1)^2\) 正確。改作：以散文陳述減號（避免列印數學內 \color），math 不變。RESOLVED none

Example 2.23　Quotient rule with exponential選收

sec-2-5.html，Example 2.22 之後、§2.5 收尾散文之前 ・ [source: CLP-1 §2.7 prob_s2.7 Procedural "f(x)=e^x/(2x)"]（CC BY-NC-SA 4.0）・解材：official（改寫）

填 L2-3（延續）：\(e^x\) 在分子、多項式在分母（與 Example 2.22(c) 的 \(x^2/e^x\) 反向），並讀 \(f'\) 正負定增減。

Differentiate \(f(x) = \dfrac{e^{x}}{2x}\).

\[ f'(x) = \frac{(e^{x})(2x) - (e^{x})(2)}{(2x)^{2}} = \frac{2e^{x}(x-1)}{4x^{2}} = \frac{(x-1)e^{x}}{2x^{2}}. \]

The top's derivative is again \(e^{x}\) (Theorem 2.5); the bottom's is \(2\). The sign of \(f'\) changes at \(x = 1\): \(f' < 0\) on \(0 < x < 1\) (decreasing), \(f' > 0\) for \(x > 1\) (increasing).

稽核 ✓ final answer \((x-1)e^x/(2x^2)\) 正確、增減判讀正確、Theorem 2.5 引用解析。RESOLVED none
