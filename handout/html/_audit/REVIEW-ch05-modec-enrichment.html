§5.8 Curve Sketching

已飽和（近滿）：opener（各層疊出形狀）＋Strategy 5.6 七步 checklist＋「cancelled factor 是 hole 不是 asymptote」已折進 Def 5.7 後散文＋兩個全套 sketch（含 slant）。缺口＝「asymptote 可穿過」的澄清。

high5.8-1caution
locus：after Definition 5.7（與其後 “Vertical-asymptote candidates…” 散文段相鄰），unnumbered Caution。
澄清「asymptote 是終局行為的描述、不是圍欄」：horizontal／slant asymptote 可以被曲線穿過——甚至無限多次，如 \(y = \dfrac{\sin x}{x}\) 每隔 \(\pi\) 就穿過其 horizontal asymptote \(y=0\) 一次（由 \(\bigl|\sin x / x\bigr| \le 1/|x| \to 0\)）；定義只要求「最終貼近」，不禁止途中相交。Vertical asymptote 則不同——函數值在該處脫韁，曲線無從穿越。
why：「圖不會碰 asymptote」是教科書級常見誤解，而本節兩個 worked example（\(x^{2}/(x^{2}-1)\) 不觸 \(y=1\)、\(x+1/x\) 不觸 \(y=x\)）恰好都不穿過，誤解全程未被挑戰；全書其他章也未講過此點（已 grep 確認）。直接影響讀者日後畫 \(xe^{-x}\)、\((\ln x)/x\) 這類圖的正確性。
source：[source: OpenStax Calculus Vol.1 §4.6（“a graph … may cross a horizontal asymptote”；動筆前覆核精確位置）]；數學內容自足
