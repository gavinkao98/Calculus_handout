## § Rolle's Theorem（手稿原文，pp.11–18）

> 手稿標題即 "§ Rolle's Theorem"；其下依序為 max/min 定義、`cos x` 例、EVT「Fact」、Thm A、Rolle、MVT、monotonicity Corollary、兩個 Example。handout 標題改為 §4.4「Rolle's Theorem and the Mean Value Theorem」。

### Def 1（maximum / minimum point）

For `f(x)` defined on a domain `D`, we say that `f(x_0)` is a **maximum** of `f` on `D` if `x_0 ∈ D` and

`f(x_0) ≥ f(x)`   `∀ x ∈ D`.

（手稿以紅筆標出對偶的 **minimum**：把 `maximum` 換成 `minimum`、`≥` 換成 `≤`，即 `f(x_0) ≤ f(x) ∀ x ∈ D`。）

### Ex（`cos x`）

For `f(x) = cos x`, `x ∈ ℝ`, we see that `0 ∈ ℝ` and

`1 = cos 0 ≥ cos x`,   `∀ x ∈ ℝ`.

Hence, `0` is a **maximum point** of `cos x` and the **maximum value** is `1`.

### Fact（Extreme Value Theorem，陳述不證）

If `f` is a continuous function defined on a closed interval `[a, b]`, then `f` has a maximum point `X_M ∈ [a, b]` and a minimum point `X_m ∈ [a, b]`.

> 手稿括註原文：「This fact can be derived from the completeness of real numbers. The whole argument is lengthy. We do not prove it for the moment.」——**明言不證**（依完備性，論證冗長）。

### Thm A（interior extremum ⟹ `f' = 0`）

**Suppose** `X_M ∈ (a, b)` is a maximum point of a continuous function `f(x)` defined on `[a, b]` and `f'(X_M)` exists. **Then** `f'(X_M) = 0`.

（手稿以紅筆標對偶 minimum 版：`X_m ∈ (a, b)`、minimum、`f'(X_m)`、`f'(X_m) = 0`。）

**證（手稿）：**
Since `f(X_M) ≥ f(x)` for all `x ∈ [a, b]`,

`f(X_M + h) − f(X_M) ≤ 0`.

⟹

`(f(X_M + h) − f(X_M)) / h ≤ 0`   for `h > 0`,

and

`(f(X_M + k) − f(X_M)) / k ≥ 0`   for `k < 0`.

Hence,

`f'(X_M) = lim_{h→0⁺, h>0} (f(X_M + h) − f(X_M)) / h ≤ 0`,

and

`f'(X_M) = lim_{k→0⁻, k<0} (f(X_M + k) − f(X_M)) / k ≥ 0`.

This implies `f'(X_M) = 0`.

> 手稿只顯式做 maximum case；minimum case 由紅筆對偶標注，視為平行（把 `≥`↔`≤`、`h>0`↔`k<0` 對調）。`h>0` 時分子 `≤0`、分母 `>0` ⟹ 商 `≤0`；`k<0` 時分子 `≤0`、分母 `<0` ⟹ 商 `≥0`。兩單邊極限一個 `≤0` 一個 `≥0`，逼出 `=0`。

### Rolle's Thm

**Suppose** `f` is continuous on `[a, b]` and is differentiable on `(a, b)`. **If** `f(a) = f(b)`, **then** there exists a point `c ∈ (a, b)` s.t. `f'(c) = 0`.

**證（手稿）：**
Suppose `X_m, X_M ∈ [a, b]` s.t.

`m = f(X_m) ≤ f(x) ≤ f(X_M) = M`   `∀ x ∈ [a, b]`.

（`X_m, X_M` 由 Fact〔EVT〕保證存在。）

**Case 1.** If `m = M = f(a) = f(b)`, then `f(x) = f(a)` for all `x ∈ (a, b)` and `f'(c) = 0` `∀ c ∈ (a, b)`.

**Case 2.** If Case 1 does not hold, then either

(i) `f(X_M) > f(a)`   or   (ii) `f(X_m) < f(a)`.

**[請查核]**：手稿 p.14 的 (ii) 寫成「`f(X_M) < f(a)`」——下標應為 `X_m`（手稿筆誤）。p.15 解 (ii) 時用的是 `X_m`（「`X_m ∈ (a,b)`、`c = X_m`」），且 legacy `cor`/Rolle 證（line 612）亦作 `f(x_m) < f(a)`。故此處取 **`f(X_m) < f(a)`**（min 落在端點值之下）。

If (i) holds, `X_M ∈ (a, b)`, and we may choose `c = X_M` so that

`f'(c) = f'(X_M) = 0`.

If (ii) holds, then `X_m ∈ (a, b)` and we may choose `c = X_m`. We then see that

`f'(c) = f'(X_m) = 0`.

This completes the proof.

> 邏輯：Case 2 中 `m < M`（否則退回 Case 1）；又 `f(a)=f(b)`，故 max 與 min 不能都等於端點值——至少一個嚴格在端點值之外，那一個（極值點）必在開區間 `(a,b)` 內部，由 Thm A 得 `f'=0`。

### The mean value theorem（MVT）

**Suppose** `f` is continuous on `[a, b]` and is differentiable on `(a, b)`. **Then** there exists a point `c ∈ (a, b)` s.t.

`f'(c) = (f(b) − f(a)) / (b − a)`.

> 手稿一句：「We shall use the Rolle's theorem to prove the mean value theorem.」

**證（手稿）：**
Set

`g(x) = f(x) − [ f(a) + ((f(b) − f(a)) / (b − a)) (x − a) ]`.

Observe: Since `f` is differentiable on `(a, b)` and continuous on `[a, b]`, so is `g(x)`.

`g(a) = 0 = g(b)`.

By the Rolle's theorem, `∃ c ∈ (a, b)` s.t.

`0 = g'(c) = f'(c) − (f(b) − f(a)) / (b − a)`,

which implies

`f'(c) = (f(b) − f(a)) / (b − a)`.

This completes the proof of the mean value theorem.

> 中括號 `[ f(a) + ((f(b)−f(a))/(b−a))(x−a) ]` 是過 `(a, f(a))`、`(b, f(b))` 的割線；`g` 是曲線與割線的垂直差。`g(a)=f(a)−f(a)=0`、`g(b)=f(b)−f(b)=0`（手稿直接給 `g(a)=0=g(b)`，未展開兩步）；`g'(x)=f'(x)−(f(b)−f(a))/(b−a)`（中括號對 `x` 微分，常數項 `f(a)` 消失、線性項斜率為割線斜率）。

### Corollary（`f' ≥ 0 ⟹ increasing`）

**Suppose** `f` is continuous on `[a, b]` and is differentiable on `(a, b)`. **If** `f'(c) ≥ 0` for all `c ∈ (a, b)`, **then**

`f(x_2) ≥ f(x_1)`   for all `a ≤ x_1 < x_2 ≤ b`.

**證（手稿）：**
Applying the mean value theorem on `[x_1, x_2]`, `∃ c ∈ (x_1, x_2)` s.t.

`(f(x_2) − f(x_1)) / (x_2 − x_1) = f'(c) ≥ 0`.

Hence `f(x_2) ≥ f(x_1)`.

### Example

**(i)** `d/dx sin x = cos x > 0` for `x ∈ [0, π/4]`.
Hence, `sin x_2 > sin x_1` for `0 ≤ x_1 < x_2 ≤ π/4`.

**(2)** `d/dx e^x = e^x > 0` for all `x ∈ ℝ`.
Hence, `e^{x_2} > e^{x_1}` for all `x_1 < x_2`.

`[ e^x is strictly increasing for all x ∈ ℝ ]`   （手稿以紅筆方括註，標重點）。

> **[請查核]（strict vs non-strict）**：手稿 Corollary 只陳述 `f' ≥ 0 ⟹ f(x_2) ≥ f(x_1)`（**非嚴格**），但兩個 Example 都下 **strict**（`sin x_2 > sin x_1`、`e^{x_2} > e^{x_1}`、"strictly increasing"）。手稿是直接重跑 MVT 商式 `(f(x_2)−f(x_1))/(x_2−x_1)=f'(c)`：當 `f'(c) > 0`（嚴格正）時分子嚴格正，得 strict。即 strict 結論來自「`f' > 0`」而非那條非嚴格 Corollary。**legacy 把 Corollary 寫成雙條（`f'≥0 ⟹` non-decreasing；`f'>0 ⟹` strictly increasing）**（line 705–710）——這是乾淨補法，且 §4.5 需要「`e^x` strictly increasing」（單射）才能定義 `ln x`。屬 ④ 方向（建議 Corollary 補 strict 版；見 ② brief §C）。

## 承重直覺（一節一個，領頭）
**MVT 是「逐點導數資訊」通往「整段函數行為」的橋——而且是一條 existence 定理（只保證有個 `c`、不告訴你 `c` 在哪）。** 你不能光盯著導數值就斷言「`f'≥0 ⟹ f` 遞增」：導數是**一點**的資訊，遞增是**一整段**的性質，中間缺一座橋。MVT 就是那座橋：`(f(b)−f(a))/(b−a)=f'(c)` 把「兩端點的平均變化率」釘在「某內部點的瞬時變化率」上。幾何上——**某條內部切線必與割線平行**（§C-4 圖）。而整個證明是一招漂亮的化約階梯：**MVT「減掉割線」化約成 Rolle（兩端等高）→ Rolle 化約成「內部極值點導數為 0」（Thm A）→ Thm A 是雙單邊極限一逼即得**。三條定理像三節階梯，每節踩在前一節上。**這座橋一旦架好，立刻收割：`f'≥0 ⟹ 遞增`（Corollary），再套到 `e^x`（`(e^x)'=e^x>0`）就得 `e^x` 嚴格遞增——正是 §4.5 定義 `ln x` 所缺的最後一塊。** existence（不指明 `c`）這點要先打臉天真期待：MVT 不是給你 `c` 的公式，是保證 `c` 存在，後續論證**只能用存在性、不能用某個特定 `c`**（§C-5 strategy 第 3 條）。

<article class="sec" lang="en">

  <header class="sec-head">
    <h2 class="sec-title"><span class="sec-no">4.4</span>Rolle's Theorem and the Mean Value Theorem</h2>
  </header>

  <!-- expansion:intuition — section opener: pivot from §4.3's closing fence (monotonicity needs an existence theorem). Frame the MVT as the bridge from pointwise derivative to interval behaviour. -->
  <p>Section 4.3 ended with a promissory note: \(e^{x}\) has positive derivative everywhere, so it <em>ought</em> to be increasing, and once we know that we can invert it to build \(\ln x\). But "positive derivative" is information at a single point, while "increasing" is a statement about the function across a whole interval, and nothing built so far bridges the two. This section constructs that bridge. Its keystone is the <em>Mean Value Theorem</em> (MVT), the central existence theorem of differential calculus: on any interval where a function is continuous and differentiable inside, some interior point carries an instantaneous slope equal to the average slope across the interval. We reach it in two steps — first <em>Rolle's theorem</em>, the special case where the average slope is zero, then the MVT itself — after one preparatory result about where maxima and minima can sit.</p>

  <!-- expansion:history [source: standard calculus-textbook historical note] — one-sentence attribution; not in the manuscript -->
  <p>The special case is named for Michel Rolle, who stated it for polynomials in 1691; the general theorem and its modern proof belong to the nineteenth-century analysts, Cauchy among them, who made the derivative rigorous.</p>

  <h3 class="subsec-head">Maxima, minima, and the extreme value theorem</h3>

  <section class="env env-definition">
    <p class="env-head"><span class="env-kicker">Definition</span><span class="env-num">4.3</span></p>
    <div class="env-body">
      <p>Let \(f\) be defined on a domain \(D \subseteq \mathbb{R}\), and let \(x_{0} \in D\). We call \(x_{0}\) a <em>maximum point</em> of \(f\) on \(D\) (and \(f(x_{0})\) the <em>maximum value</em>) if</p>
      <p>\[ f(x_{0}) \ge f(x) \qquad \text{for every } x \in D, \]</p>
      <p>and a <em>minimum point</em> (with <em>minimum value</em> \(f(x_{0})\)) if instead \(f(x_{0}) \le f(x)\) for every \(x \in D\).</p>
      <p class="informal">A maximum point is where \(f\) attains its largest value on \(D\), a minimum point where it attains its smallest. A function may have several maximum points sharing one maximum value, or none at all when \(D\) is unbounded or open and \(f\) runs off to \(\pm\infty\).</p>
    </div>
  </section>

  <p>For instance, \(x_{0} = 0\) is a maximum point of \(\cos x\) on \(\mathbb{R}\), with maximum value \(1\), since \(1 = \cos 0 \ge \cos x\) for every real \(x\).</p>

  <p>Whether a function attains its extremes at all is a delicate matter on a general domain, but on a closed interval the answer is always yes. This is the foundational fact that drives everything below.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.9</span><span class="env-name">Extreme Value Theorem</span></p>
    <div class="env-body">
      <p>If \(f\) is continuous on a closed interval \([a, b]\), then \(f\) attains both a maximum and a minimum on \([a, b]\): there exist points \(x_{M}, x_{m} \in [a, b]\) with</p>
      <p>\[ f(x_{m}) \le f(x) \le f(x_{M}) \qquad \text{for every } x \in [a, b]. \]</p>
    </div>
  </section>

  <!-- expansion:intuition — the manuscript states this Fact without proof (completeness); record why, and that both hypotheses are essential -->
  <p>We state this without proof. The argument rests on the completeness of \(\mathbb{R}\) (Theorem 4.1) and is the kind of lengthy analytic construction we set aside at this level; the theorem is used here as a black box. Both hypotheses are genuinely needed: on the <em>open</em> interval \((0, 1)\) the function \(f(x) = x\) attains no maximum — its values approach \(1\) without ever reaching it — and a discontinuous function can miss its extremes even on a closed interval. We will only ever apply the theorem to continuous functions on closed intervals, where it always delivers.</p>

  <h3 class="subsec-head">Interior extrema: Theorem A</h3>

  <p>The differential-calculus content is the next theorem: when an extreme value is attained strictly <em>inside</em> the interval — not at an endpoint — the derivative there must vanish.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.10</span><span class="env-name">Interior extremum (Theorem A)</span></p>
    <div class="env-body">
      <p>Suppose \(f\) is defined on \([a, b]\), the point \(x_{M} \in (a, b)\) is a maximum (or minimum) point of \(f\) on \([a, b]\), and \(f'(x_{M})\) exists. Then \(f'(x_{M}) = 0\).</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Take \(x_{M}\) to be a maximum point; the minimum case is identical with the inequalities reversed. Since \(f(x_{M}) \ge f(x)\) for all \(x \in [a, b]\),</p>
      <p>\[ f(x_{M} + h) - f(x_{M}) \le 0 \]</p>
      <p>for every increment \(h\) small enough that \(x_{M} + h\) stays in \([a, b]\). Because \(x_{M}\) lies in the open interior \((a, b)\), both small positive and small negative \(h\) are allowed. Dividing by \(h\) and watching the sign of the denominator:</p>
      <p>\[ \frac{f(x_{M} + h) - f(x_{M})}{h} \le 0 \quad (h > 0), \qquad \frac{f(x_{M} + h) - f(x_{M})}{h} \ge 0 \quad (h < 0). \]</p>
      <p>The derivative \(f'(x_{M})\) exists, so the one-sided limits both equal it. Letting \(h \to 0^{+}\) through positive values keeps the quotient \(\le 0\), and letting \(h \to 0^{-}\) through negative values keeps it \(\ge 0\):</p>
      <p>\[ f'(x_{M}) = \lim_{h \to 0^{+}} \frac{f(x_{M} + h) - f(x_{M})}{h} \le 0, \qquad f'(x_{M}) = \lim_{h \to 0^{-}} \frac{f(x_{M} + h) - f(x_{M})}{h} \ge 0. \]</p>
      <p>A single number that is both \(\le 0\) and \(\ge 0\) is \(0\), so \(f'(x_{M}) = 0\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <!-- expansion:caution — the "interior" hypothesis is essential; endpoint extrema need not have zero derivative -->
  <section class="env env-caution">
    <p class="env-head"><span class="env-kicker">Caution</span></p>
    <div class="env-body">
      <p>Theorem 4.10 needs \(x_{M}\) in the <em>open</em> interior \((a, b)\). At an endpoint the conclusion can fail: \(f(x) = x\) on \([0, 1]\) takes its minimum at \(x = 0\) and its maximum at \(x = 1\), yet \(f'(x) = 1 \ne 0\) everywhere. An endpoint offers only a one-sided increment, so the two-sided squeeze that forced \(f'(x_{M}) = 0\) is unavailable.</p>
    </div>
  </section>

  <h3 class="subsec-head">Rolle's theorem</h3>

  <p>Rolle's theorem marries the two results above. The picture: if a function leaves and returns to the same height over \([a, b]\), it must turn around somewhere inside, and at that interior turning point — an extremum — the derivative is zero.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.11</span><span class="env-name">Rolle's theorem</span></p>
    <div class="env-body">
      <p>Suppose \(f\) is continuous on \([a, b]\) and differentiable on \((a, b)\), with \(f(a) = f(b)\). Then there exists a point \(c \in (a, b)\) such that \(f'(c) = 0\).</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>By the extreme value theorem (Theorem 4.9), \(f\) attains a maximum at some \(x_{M} \in [a, b]\) and a minimum at some \(x_{m} \in [a, b]\), with \(f(x_{m}) \le f(x) \le f(x_{M})\) for all \(x \in [a, b]\). Write \(m = f(x_{m})\) and \(M = f(x_{M})\).</p>
      <p><strong>Case 1: \(m = M\).</strong> Together with \(f(a) = f(b)\) this forces \(m = M = f(a) = f(b)\), so the bounds \(m \le f(x) \le M\) collapse and \(f\) is constant on \([a, b]\). Then \(f'(c) = 0\) for every \(c \in (a, b)\); choose any.</p>
      <p><strong>Case 2: \(m &lt; M\).</strong> Since \(f(a) = f(b)\), the values \(m\) and \(M\) cannot both equal that common endpoint value, so at least one of</p>
      <p>\[ f(x_{M}) > f(a) \qquad \text{or} \qquad f(x_{m}) < f(a) \]</p>
      <p>holds. If \(f(x_{M}) > f(a) = f(b)\), then \(x_{M}\) is neither endpoint, so \(x_{M} \in (a, b)\); by Theorem 4.10, \(f'(x_{M}) = 0\), and we set \(c = x_{M}\). Otherwise \(f(x_{m}) < f(a) = f(b)\), the same argument places \(x_{m} \in (a, b)\), and Theorem 4.10 gives \(f'(x_{m}) = 0\) with \(c = x_{m}\). Either way an interior \(c\) with \(f'(c) = 0\) exists. <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <h3 class="subsec-head">The Mean Value Theorem</h3>

  <p>The MVT lifts the equal-endpoints restriction. Allow any two endpoint values, and the conclusion becomes that the derivative somewhere equals the <em>average</em> rate of change \((f(b) - f(a))/(b - a)\) — the slope of the secant joining the two endpoints.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.12</span><span class="env-name">Mean Value Theorem</span></p>
    <div class="env-body">
      <p>Suppose \(f\) is continuous on \([a, b]\) and differentiable on \((a, b)\). Then there exists a point \(c \in (a, b)\) such that</p>
      <p>\[ f'(c) = \frac{f(b) - f(a)}{b - a}. \]</p>
    </div>
  </section>

  <!-- expansion:figure — the canonical MVT picture (ROADMAP §4.4 key figure); registered in chapter4 FIGS as "mvt-secant-tangent" -->
  <figure class="figure" data-fig="mvt-secant-tangent">
    <figcaption><span class="fig-no">Figure 4.2</span> The Mean Value Theorem. The secant through \((a, f(a))\) and \((b, f(b))\) (dashed) has slope \((f(b) - f(a))/(b - a)\); the theorem guarantees an interior point \(c\) where the tangent (solid) is parallel to it. The proof tilts the picture by subtracting the secant, reducing the claim to Rolle's theorem.</figcaption>
  </figure>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Subtract the secant line from \(f\). Define the auxiliary function</p>
      <p>\[ g(x) = f(x) - \left[ f(a) + \frac{f(b) - f(a)}{b - a} (x - a) \right]. \]</p>
      <p>The bracketed term is exactly the secant line through \((a, f(a))\) and \((b, f(b))\), so \(g(x)\) measures the vertical gap between the curve and that secant. It inherits the hypotheses of Rolle's theorem:</p>
      <ul>
        <li>\(g\) is continuous on \([a, b]\) and differentiable on \((a, b)\), being \(f\) minus a linear function;</li>
        <li>\(g(a) = f(a) - f(a) = 0\) and \(g(b) = f(b) - f(b) = 0\), so \(g(a) = g(b)\).</li>
      </ul>
      <p>By Rolle's theorem (Theorem 4.11) there is a \(c \in (a, b)\) with \(g'(c) = 0\). Differentiating \(g\), the linear bracket contributes its constant slope, so</p>
      <p>\[ 0 = g'(c) = f'(c) - \frac{f(b) - f(a)}{b - a}, \]</p>
      <p>which rearranges to \(f'(c) = (f(b) - f(a))/(b - a)\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <!-- expansion:intuition — Remark 4.4 (§C-10, optional taken): the existence-engine synthesis tying Theorem A → Rolle → MVT and pointing forward -->
  <section class="env env-remark">
    <p class="env-head"><span class="env-kicker">Remark</span><span class="env-num">4.4</span></p>
    <div class="env-body">
      <p>Rolle's theorem is precisely the MVT in the case \(f(a) = f(b)\), where the average slope is \(0\) — yet we proved Rolle <em>first</em> and recovered the MVT from it, because subtracting the secant always tilts the general situation back to level endpoints. The three results form a ladder: Theorem A, a statement about the derivative at one point, supports Rolle; Rolle adds the equal-endpoints hypothesis, and Rolle in turn supports the MVT for arbitrary endpoints. The MVT is the load-bearing rung. It is the standard device for turning pointwise information about \(f'\) into behaviour of \(f\) across an interval, and it returns every time we make that move — immediately below to read monotonicity off the sign of \(f'\), and later for Taylor's theorem and the error estimates that lean on it.</p>
    </div>
  </section>

  <!-- expansion:strategy — Strategy 4.2 (§C-5, ROADMAP): verifying MVT hypotheses; |x| counterexample + closed/open asymmetry folded in (§C-6b) -->
  <section class="env env-strategy">
    <p class="env-head"><span class="env-kicker">Strategy</span><span class="env-num">4.2</span><span class="env-name">Applying the Mean Value Theorem</span></p>
    <div class="env-body">
      <p>To extract information about \(f'\) on an interval \([a, b]\) from the MVT:</p>
      <ol class="steps">
        <li>Check that \(f\) is continuous on the <em>closed</em> interval \([a, b]\) — endpoints included. Continuity at the endpoints is what lets the proof call on the extreme value theorem, which needs a closed interval.</li>
        <li>Check that \(f\) is differentiable on the <em>open</em> interval \((a, b)\) — endpoints excluded. The endpoints need no derivative, since Rolle's theorem only ever touched the derivative at an interior point. (So \(f(x) = \sqrt{x}\) on \([0, 1]\), differentiable on \((0, 1)\) despite its vertical tangent at \(0\), is fair game.)</li>
        <li>Read off the conclusion: there <em>exists</em> a \(c \in (a, b)\) with \(f'(c) = (f(b) - f(a))/(b - a)\). The theorem promises existence only; it does not say which \(c\), so any further argument may rely on the existence of such a \(c\), but not on any specific value.</li>
      </ol>
      <p>Skipping the checks is the usual mistake, and the conclusion can then be false. Take \(\lvert x \rvert\) on \([-1, 1]\): it is continuous everywhere but not differentiable at \(0\), so step 2 fails. The secant slope is \((\lvert 1 \rvert - \lvert -1 \rvert)/(1 - (-1)) = 0\), yet \(\lvert x \rvert' = \pm 1\) wherever it is defined and never \(0\) — there is no valid \(c\), exactly because the differentiability hypothesis was missing.</p>
    </div>
  </section>

  <!-- expansion:example — Example 4.2 (optional taken): a direct MVT application where c can be found explicitly -->
  <div class="workedexample">
    <section class="env env-example">
      <p class="env-head"><span class="env-kicker">Example</span><span class="env-num">4.2</span></p>
      <div class="env-body">
        <p>Apply the Mean Value Theorem to \(f(x) = x^{2}\) on \([0, 2]\), and locate a point \(c\) it guarantees.</p>
      </div>
    </section>
    <section class="env env-solution">
      <p class="env-head"><span class="env-kicker">Solution</span></p>
      <div class="env-body">
        <p>The function \(x^{2}\) is continuous on \([0, 2]\) and differentiable on \((0, 2)\), so both hypotheses hold. The average rate of change is</p>
        <p>\[ \frac{f(2) - f(0)}{2 - 0} = \frac{4 - 0}{2} = 2. \]</p>
        <p>The MVT promises a \(c \in (0, 2)\) with \(f'(c) = 2\). Here \(f'(x) = 2x\), so \(2c = 2\) gives \(c = 1\), which indeed lies in \((0, 2)\). For this simple \(f\) we could solve for \(c\) outright; in general the theorem hands us only the existence of such a \(c\), and the arguments below are built to need nothing more. <span class="qed"></span></p>
      </div>
    </section>
  </div>

  <h3 class="subsec-head">Monotonicity from the sign of the derivative</h3>

  <p>Now the payoff. The MVT converts the sign of the derivative directly into monotonic behaviour, turning the qualitative slogan "positive derivative means increasing" into a theorem.</p>

  <!-- expansion:formula — Corollary 4.3 (§C-2): both forms stated; the strict form is what §4.5 needs (the manuscript states only the non-strict form, yet its examples conclude strict) -->
  <section class="env env-corollary">
    <p class="env-head"><span class="env-kicker">Corollary</span><span class="env-num">4.3</span><span class="env-name">Monotonicity from the sign of \(f'\)</span></p>
    <div class="env-body">
      <p>Suppose \(f\) is continuous on \([a, b]\) and differentiable on \((a, b)\).</p>
      <ul>
        <li>If \(f'(c) \ge 0\) for every \(c \in (a, b)\), then \(f(x_{1}) \le f(x_{2})\) whenever \(a \le x_{1} < x_{2} \le b\): \(f\) is non-decreasing on \([a, b]\).</li>
        <li>If \(f'(c) > 0\) for every \(c \in (a, b)\), then \(f(x_{1}) < f(x_{2})\) whenever \(a \le x_{1} < x_{2} \le b\): \(f\) is strictly increasing on \([a, b]\).</li>
      </ul>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Fix \(x_{1} < x_{2}\) in \([a, b]\). The hypotheses hold on the subinterval \([x_{1}, x_{2}]\) as well, so by the MVT (Theorem 4.12) there is a \(c \in (x_{1}, x_{2})\) with</p>
      <p>\[ \frac{f(x_{2}) - f(x_{1})}{x_{2} - x_{1}} = f'(c). \]</p>
      <p>The denominator \(x_{2} - x_{1}\) is positive. If \(f'(c) \ge 0\) the numerator \(f(x_{2}) - f(x_{1})\) is \(\ge 0\), giving \(f(x_{1}) \le f(x_{2})\); if \(f'(c) > 0\) it is strictly positive, giving \(f(x_{1}) < f(x_{2})\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <p>Two applications close the section — the second is the one we have been working toward.</p>

  <div class="workedexample">
    <section class="env env-example">
      <p class="env-head"><span class="env-kicker">Example</span><span class="env-num">4.3</span></p>
      <div class="env-body">
        <p>Show that \(\sin x\) is strictly increasing on \([0, \tfrac{\pi}{4}]\).</p>
      </div>
    </section>
    <section class="env env-solution">
      <p class="env-head"><span class="env-kicker">Solution</span></p>
      <div class="env-body">
        <p>On \((0, \tfrac{\pi}{4})\) the derivative \((\sin x)' = \cos x\) is positive, since cosine is positive on \((0, \tfrac{\pi}{2})\). By the strict half of Corollary 4.3, \(\sin x\) is strictly increasing on \([0, \tfrac{\pi}{4}]\): for \(0 \le x_{1} < x_{2} \le \tfrac{\pi}{4}\), \(\sin x_{1} < \sin x_{2}\). <span class="qed"></span></p>
      </div>
    </section>
  </div>

  <div class="workedexample">
    <section class="env env-example">
      <p class="env-head"><span class="env-kicker">Example</span><span class="env-num">4.4</span></p>
      <div class="env-body">
        <p>Show that \(e^{x}\) is strictly increasing on all of \(\mathbb{R}\).</p>
      </div>
    </section>
    <section class="env env-solution">
      <p class="env-head"><span class="env-kicker">Solution</span></p>
      <div class="env-body">
        <p>By Theorem 4.8, \((e^{x})' = e^{x}\). This derivative is positive everywhere: for \(x > 0\) every term of \(\sum x^{n}/n!\) is positive, at \(x = 0\) the sum is \(e^{0} = 1 > 0\), and for \(x < 0\) the exponent law (Theorem 4.7) gives \(e^{x} = 1/e^{-x} > 0\). So on any closed interval \([x_{1}, x_{2}]\) the strict half of Corollary 4.3 applies and yields \(e^{x_{1}} < e^{x_{2}}\).</p>
        <!-- expansion:intuition — §C-6c: the every-[a,b] ⟹ all-of-ℝ extension, noted in one line -->
        <p>Because any two reals \(x_{1} < x_{2}\) lie in some such closed interval, this holds for every pair, and \(e^{x}\) is strictly increasing on \(\mathbb{R}\). (Strict monotonicity passes from every closed interval to the whole line this way. Not every interval property survives such a passage — boundedness, for one, can be lost — but strict monotonicity does.) <span class="qed"></span></p>
      </div>
    </section>
  </div>

  <!-- expansion:summary — close: e^x strictly increasing ⟹ injective ⟹ §4.5 can define ln x as its inverse (minimal hand-off only; §4.5's continuity/derivative program is NOT previewed — gate-2 §4.5-scope-leak fix) -->
  <p>That last example is the result the chapter has been driving at. A strictly increasing function is one-to-one, so \(e^{x}\) has a genuine inverse on its range; the next section defines that inverse, the natural logarithm \(\ln x\).</p>
