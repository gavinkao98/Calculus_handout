## [e2] e^x is a continuous function for x > 0

To show `lim_{x→x₀} e^x = e^{x₀}`, `x₀ > 0`:

For given `ε > 0`, choose `k > n₀ > 2(x₀+1)` such that
`((x₀+1)^(n₀)/n₀!)·(1/2)^(k−n₀) < ε`.

Since `P_k(x) := Σ_{n=0}^{k} xⁿ/n!` is a continuous function
[any polynomial is a continuous function], `∃ δ > 0` with `δ < min{x₀/2, 1/2}` such that
`|P_k(x) − P_k(x₀)| < ε`   provided   `|x − x₀| < δ`.

Now:
`e^x − e^{x₀} = (e^x − P_k(x)) + (P_k(x) − P_k(x₀)) + (P_k(x₀) − e^{x₀})`.

Hence, for `|x − x₀| < δ = min{x₀/2, 1/2}`,
`|e^x − e^{x₀}|`
` ≤ |e^x − P_k(x)| + |P_k(x) − P_k(x₀)| + |P_k(x₀) − e^{x₀}|`
` ≤ ((x₀+1)^(n₀)/n₀!)·(1/2)^(k−n₀)  +  ε  +  ((x₀+1)^(n₀)/n₀!)·(1/2)^(k−n₀)`
` < 3ε`.

This proves that `lim_{x→x₀} e^x = e^{x₀}`. Hence, `e^x` is a continuous function for `x > 0`.

(兩個端點項用 (∗) 控制：`x`, `x₀` 均落在 `(0, x₀+1)` 內，故 `L = x₀+1`、`n₀ > 2(x₀+1)`，兩處 `|e^· − P_k(·)|` 各 `< ε` 的尾界部分。中間項用 `P_k` 的多項式連續。)

## (△) Property — absolute convergence implies convergence

**(△) Property:** If `Σ_{n=1}^∞ |aₙ| < +∞`, then `Σ_{n=1}^∞ aₙ` converges.

**Note:** when we say `Σ_{n=1}^∞ aₙ` converges, this means `Sₖ = Σ_{n=1}^{k} aₙ` is a convergent
sequence, i.e. `lim_{k→∞} (Σ_{n=1}^{k} aₙ)` exists.

### Def — Cauchy sequence

A sequence `{aₙ}_{n=1}^∞` is called a **Cauchy sequence** if for any `ε > 0`, `∃ N₀ ∈ ℕ`
such that `|aₘ − aₙ| < ε` for any `m, n ≥ N₀`.

### Thm — convergent ⟺ Cauchy

A sequence `{aₙ}_{n=1}^∞` is a convergent sequence **if and only if** `{aₙ}_{n=1}^∞` is a
Cauchy sequence.

> **手稿明寫 punt：** "For the moment, we shall not prove this theorem."

> 證明——超出手稿、user-directed 2026-04-27。屬 ② brief／④ draft 的擴增，非本 seed 轉錄範圍。）

### Proof of (△) using Cauchy

To show (△) property, set `Sₙ = Σ_{k=1}^{n} aₖ`, where `Σ_{k=1}^∞ |aₖ| < +∞`.

Since `Σ_{k=1}^∞ |aₖ| < +∞`, for any `ε > 0`, `∃ N₀ ∈ ℕ` such that
`Σ_{k=N₀+1}^∞ |aₖ| < ε`.

Therefore for `m > n > N₀`,
`|Sₘ − Sₙ| ≤ Σ_{k=n+1}^{m} |aₖ| ≤ Σ_{k=N₀+1}^∞ |aₖ| < ε`.

This proves that `{Sₙ}_{n=1}^∞` is a Cauchy sequence. Hence, `Σ_{k=1}^∞ aₖ` converges. ∎

## Exponent law: e^x · e^y = e^{x+y}

> 以下假設 `x, y ∈ [−L, L]`，`k > n₀ > 8L`。

### We observe that

**(1)** `|e^y| ≤ |Σ_{n=0}^∞ yⁿ/n!| ≤ Σ_{n=0}^∞ |y|ⁿ/n! ≤ e^(|y|) < +∞`,   and
`|e^y − P_k(y)| ≤ Σ_{n=k+1}^∞ |y|ⁿ/n! ≤ |y|^k/k!`   if `k > n₀ > 2|y|`.

(此步同時給出 [e3] 的有限性：`Σ|y|ⁿ/n! = e^(|y|) < ∞` ⟹ 由 (△) 級數絕對收斂 ⟹ `e^y` 對 `y<0` 有定義。)

**(2)** `P_k(x)·P_k(y) = (Σ_{n=0}^{k} xⁿ/n!)(Σ_{n=0}^{k} yⁿ/n!)`
` = Σ_{ℓ=0}^{k} Σ_{m=0}^{ℓ} [1/(m!(ℓ−m)!)] xᵐ y^(ℓ−m)`   … (I)
` + Σ_{j=1}^{k} Σ_{m=j}^{k} [1/(m!(k+j−m)!)] xᵐ y^(k+j−m)`   … (II)

### 二項式定理 (Binomial Theorem)

**Notation:** `Cⁿ_k := n!/(k!(n−k)!)`, `k, n ∈ ℕ` and `k ≤ n`.
（D8：本書全程改用 `\binom{n}{k}`；首見處 cross-ref 手稿記號 `Cⁿ_k`。）

`(x + y)ⁿ = Σ_{k=0}^{n} Cⁿ_k · xᵏ y^(n−k)`.

### (I) = P_k(x+y)

`(I) = Σ_{ℓ=0}^{k} Σ_{m=0}^{ℓ} [1/(m!(ℓ−m)!)] xᵐ y^(ℓ−m)`
`    = Σ_{ℓ=0}^{k} (1/ℓ!) Σ_{m=0}^{ℓ} [ℓ!/(m!(ℓ−m)!)] xᵐ y^(ℓ−m)`
`    = Σ_{ℓ=0}^{k} (1/ℓ!) Σ_{m=0}^{ℓ} C^ℓ_m xᵐ y^(ℓ−m)`
`    = Σ_{ℓ=0}^{k} (1/ℓ!) (x+y)^ℓ  =  P_k(x+y)`.

### (II) tail bound

`|(II)| = |Σ_{j=1}^{k} Σ_{m=j}^{k} [1/(m!(k+j−m)!)] xᵐ y^(k+j−m)|`
`      ≤ Σ_{ℓ=k+1}^{2k} Σ_{m=0}^{ℓ} [1/(m!(ℓ−m)!)] |x|ᵐ |y|^(ℓ−m)`
`      ≤ Σ_{ℓ=k+1}^{2k} (1/ℓ!) (|x|+|y|)^ℓ`
`      ≤ (1/k!) (|x|+|y|)^k`
`      ≤ (2L)^(n₀)/n₀! · (1/2)^(k−n₀)`.

（末步用 `|x|+|y| ≤ 2L` 與 `k > n₀ > 8L`。**[請查核]** 中間步 `Σ_{ℓ=k+1}^{2k}(1/ℓ!)(|x|+|y|)^ℓ ≤ (1/k!)(|x|+|y|)^k` 的係數收法手稿較簡略。）

Hence,
`|P_k(x)·P_k(y) − P_k(x+y)| ≤ (2L)^(n₀)/n₀! · (1/2)^(k−n₀)`.

**Note also** (同 (∗)，對 `|x| ≤ L`):
`|e^x − P_k(x)| ≤ L^k/k! ≤ (L^(n₀)/n₀!) · (1/2)^(k−n₀)`.

### Four-term split → limit

`e^x e^y − e^{x+y}`
` = (e^x − P_k(x)) e^y  +  P_k(x) (e^y − P_k(y))  +  (P_k(x) P_k(y) − P_k(x+y))  +  (P_k(x+y) − e^{x+y})`

implies
`|e^x e^y − e^{x+y}|`
` ≤ (L^(n₀)/n₀!)(1/2)^(k−n₀) · e^L`
` + e^L · (L^(n₀)/n₀!)(1/2)^(k−n₀)`
` + (2L)^(n₀)/n₀! · (1/2)^(k−n₀)`
` + (2L)^(n₀)/n₀! · (1/2)^(k−n₀)`.

By taking the limit `k → ∞`, we see that `e^x · e^y = e^{x+y}` for `x, y ∈ [−L, L]`.

Let `L → ∞`, we see that
**`e^x · e^y = e^{x+y}` holds for all `x, y ∈ ℝ`.**

  - obs (2)：`P_k(x)P_k(y) = (I) + (II)`（雙和）；
  - **二項式定理**（`Cⁿ_k:=n!/(k!(n−k)!)`、`(x+y)ⁿ=Σ Cⁿ_k xᵏ y^{n−k}`）→ `(I) = P_k(x+y)`；

  <header class="sec-head">
    <h2 class="sec-title"><span class="sec-no">4.2</span>Continuity and the Exponent Law for \(e^{x}\)</h2>
  </header>

  <!-- expansion:intuition — section opener: §4.1 gave existence for x>0; this section gives good behaviour (continuity, multiplicativity) and extends the definition to all of ℝ -->
  <p>Section 4.1 gave \(e^{x}\) an honest existence: for \(x > 0\) the defining series converges to a finite number, and the partial sums \(P_{k}(x) = \sum_{n=0}^{k} x^{n}/n!\) close in on it at a geometric rate, captured by the tail bound \((*)\). What we have <em>not</em> yet shown is that \(e^{x}\) behaves the way the symbol promises: that it is continuous, that it obeys \(e^{x} e^{y} = e^{x + y}\), and that it even makes sense for \(x \le 0\). This section settles all three. The engine throughout is a single idea: approximate the transcendental function \(e^{x}\) by the polynomial \(P_{k}(x)\), and use \((*)\) to make the error uniformly small. Once the error is controlled, the polynomial carries the argument. The one genuinely new tool we need is a convergence test that survives sign changes, since the partial sums for \(x < 0\) are no longer monotone.</p>

  <h3 class="subsec-head">Continuity for \(x > 0\)</h3>

  <p>Fix a base point \(x_{0} > 0\). We want \(\lim_{x \to x_{0}} e^{x} = e^{x_{0}}\). The difficulty is that \(e^{x}\) is an infinite series, not a polynomial; but its partial sum \(P_{k}\) <em>is</em> a polynomial, hence continuous, and the bound \((*)\) says \(P_{k}\) is uniformly close to \(e^{x}\) for every \(x\) in a fixed interval at once. The proof simply routes the comparison \(e^{x}\) versus \(e^{x_{0}}\) through \(P_{k}\) at both points.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.3</span></p>
    <div class="env-body">
      <p>The function \(e^{x}\) is continuous at every \(x_{0} > 0\).</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Let \(x_{0} > 0\) and let \(\varepsilon > 0\) be given. Work on the interval \(0 < x < x_{0} + 1\), so we may take \(L = x_{0} + 1\) in the tail bound \((*)\). Choose integers \(k > n_{0} > 2(x_{0} + 1)\) large enough that</p>
      <p>\[ \frac{(x_{0} + 1)^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}} < \varepsilon. \]</p>
      <p>This is possible because the left-hand side tends to \(0\) as \(k \to \infty\). With \(k\) now fixed, \(P_{k}(x) = \sum_{n=0}^{k} x^{n}/n!\) is a polynomial, and every polynomial is continuous, so there is a \(\delta > 0\), which we may take smaller than \(\min\{x_{0}/2,\ 1/2\}\), such that</p>
      <p>\[ \lvert P_{k}(x) - P_{k}(x_{0}) \rvert < \varepsilon \qquad \text{whenever } \lvert x - x_{0} \rvert < \delta. \]</p>
      <p>Now split the difference \(e^{x} - e^{x_{0}}\) by inserting and removing the common polynomial:</p>
      <p>\[ e^{x} - e^{x_{0}} = \bigl(e^{x} - P_{k}(x)\bigr) + \bigl(P_{k}(x) - P_{k}(x_{0})\bigr) + \bigl(P_{k}(x_{0}) - e^{x_{0}}\bigr). \]</p>
      <p>Suppose \(\lvert x - x_{0} \rvert < \delta\). Then \(x_{0} < x_{0} + 1\) and \(x < x_{0} + \tfrac{1}{2} < x_{0} + 1\), so both \(x\) and \(x_{0}\) lie in \((0,\ x_{0} + 1)\); the bound \((*)\) controls the two outer terms, and the choice of \(\delta\) controls the middle one. By the triangle inequality,</p>
      <p>\[ \lvert e^{x} - e^{x_{0}} \rvert \le \underbrace{\lvert e^{x} - P_{k}(x) \rvert}_{< \varepsilon} + \underbrace{\lvert P_{k}(x) - P_{k}(x_{0}) \rvert}_{< \varepsilon} + \underbrace{\lvert P_{k}(x_{0}) - e^{x_{0}} \rvert}_{< \varepsilon} < 3\varepsilon. \]</p>
      <p>Since \(\varepsilon > 0\) was arbitrary, \(\lim_{x \to x_{0}} e^{x} = e^{x_{0}}\). Thus \(e^{x}\) is continuous at every \(x_{0} > 0\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <p>The whole proof turned on one feature of \((*)\): it is <em>uniform</em>. The same fixed polynomial \(P_{k}\) approximates \(e^{x}\) at \(x\) and at \(x_{0}\) simultaneously, so the two outer errors are small together. We will lean on exactly this uniformity again when we multiply two exponentials.</p>

  <h3 class="subsec-head">A convergence test that survives sign changes</h3>

  <!-- expansion:intuition — why x<0 needs a new tool (Remark 4.2): the monotone-bounded argument of §4.1 fails for alternating partial sums -->
  <section class="env env-remark">
    <p class="env-head"><span class="env-kicker">Remark</span><span class="env-num">4.2</span></p>
    <div class="env-body">
      <p>For \(x < 0\) the terms \(x^{n}/n!\) alternate in sign, so the partial sums no longer increase steadily, and the monotone-bounded argument behind Theorem 4.2 simply does not apply. We need a convergence criterion that ignores signs. The natural one is: if the series of <em>absolute values</em> converges, the original series should converge too — its terms cannot accumulate any net oscillation that the absolute series has already shown to be summable.</p>
    </div>
  </section>

  <p>We record this principle, then build the machinery that proves it. The machinery — Cauchy sequences and the Bolzano–Weierstrass theorem — is of independent importance and will be reused whenever convergence must be established without an explicit limit in hand.</p>

  <section class="env env-proposition">
    <p class="env-head"><span class="env-kicker">Proposition</span><span class="env-num">4.1</span><span class="env-name">Absolute convergence implies convergence</span></p>
    <div class="env-body">
      <p>Let \(\{a_{n}\}_{n=1}^{\infty}\) be a sequence of real numbers. If \(\displaystyle\sum_{n=1}^{\infty} \lvert a_{n} \rvert < \infty\), then \(\displaystyle\sum_{n=1}^{\infty} a_{n}\) converges.</p>
    </div>
  </section>

  <p>To say that \(\sum_{n=1}^{\infty} a_{n}\) <em>converges</em> means precisely that its sequence of partial sums \(S_{k} = \sum_{n=1}^{k} a_{n}\) has a limit, i.e. \(\lim_{k \to \infty} S_{k}\) exists. The trouble is that for a series with mixed signs we have no candidate value for that limit and no monotonicity to fall back on. The way out is to recognize convergence from the <em>internal</em> behavior of the partial sums alone.</p>

  <section class="env env-definition">
    <p class="env-head"><span class="env-kicker">Definition</span><span class="env-num">4.2</span></p>
    <div class="env-body">
      <p>A sequence \(\{a_{n}\}_{n=1}^{\infty}\) is a <em>Cauchy sequence</em> if for every \(\varepsilon > 0\) there is an \(N_{0} \in \mathbb{N}\) such that</p>
      <p>\[ \lvert a_{m} - a_{n} \rvert < \varepsilon \qquad \text{for all } m, n \ge N_{0}. \]</p>
      <p class="informal">Informally, the terms eventually huddle together: past some point, any two of them differ by less than \(\varepsilon\). The definition mentions no limit value — only the terms’ closeness to one another.</p>
    </div>
  </section>

  <!-- expansion:intuition — D3① (user-directed): the manuscript writes "we shall not prove this theorem" for convergent⟺Cauchy; the handout supplies the full proof, which rests on Bolzano–Weierstrass, itself proved from completeness via the monotone-subsequence peak argument (ROADMAP resolved 2026-04-27) -->
  <p>That a sequence converges exactly when it is Cauchy is the bridge we need: it lets us certify the convergence of \(\sum a_{n}\) without naming the sum. The manuscript states this equivalence and uses it directly. We instead prove it, because the proof exposes a dependency worth seeing — the equivalence is, in the end, the completeness of \(\mathbb{R}\) (Theorem 4.1) wearing a different costume. The hinge is a classical existence theorem.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.4</span><span class="env-name">Bolzano–Weierstrass</span></p>
    <div class="env-body">
      <p>Every bounded sequence of real numbers has a convergent subsequence.</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>We first extract a <em>monotone</em> subsequence, then invoke completeness. Call an index \(p\) a <em>peak</em> of the sequence \(\{a_{n}\}\) if \(a_{p} \ge a_{n}\) for every \(n > p\) — that is, no later term ever exceeds \(a_{p}\). There are two cases.</p>
      <p class="para-head">Infinitely many peaks.</p>
      <p>List the peaks in increasing order \(p_{1} < p_{2} < p_{3} < \cdots\). By the defining property of a peak, \(a_{p_{1}} \ge a_{p_{2}} \ge a_{p_{3}} \ge \cdots\), so \(\{a_{p_{j}}\}\) is a non-increasing subsequence. It is bounded below (the whole sequence is bounded), so by part (2) of Theorem 4.1 it converges.</p>
      <p class="para-head">Only finitely many peaks.</p>
      <p>Let \(p_{*}\) be larger than every peak, so no index \(\ge p_{*}\) is a peak. Pick any \(n_{1} \ge p_{*}\). Since \(n_{1}\) is not a peak, some \(n_{2} > n_{1}\) has \(a_{n_{2}} > a_{n_{1}}\); since \(n_{2}\) is not a peak, some \(n_{3} > n_{2}\) has \(a_{n_{3}} > a_{n_{2}}\); continuing, we obtain a strictly increasing subsequence \(a_{n_{1}} < a_{n_{2}} < a_{n_{3}} < \cdots\). It is bounded above, so by part (1) of Theorem 4.1 it converges.</p>
      <p>In either case the bounded sequence has a convergent (indeed monotone) subsequence. <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <p>With Bolzano–Weierstrass in hand, the Cauchy criterion follows.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.5</span><span class="env-name">Cauchy criterion</span></p>
    <div class="env-body">
      <p>A sequence \(\{a_{n}\}_{n=1}^{\infty}\) of real numbers converges if and only if it is a Cauchy sequence.</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p class="para-head">Convergent \(\Rightarrow\) Cauchy.</p>
      <p>Suppose \(a_{n} \to A\). Given \(\varepsilon > 0\), choose \(N_{0}\) with \(\lvert a_{n} - A \rvert < \varepsilon/2\) for all \(n \ge N_{0}\). Then for \(m, n \ge N_{0}\),</p>
      <p>\[ \lvert a_{m} - a_{n} \rvert \le \lvert a_{m} - A \rvert + \lvert A - a_{n} \rvert < \tfrac{\varepsilon}{2} + \tfrac{\varepsilon}{2} = \varepsilon, \]</p>
      <p>so the sequence is Cauchy.</p>
      <p class="para-head">Cauchy \(\Rightarrow\) convergent.</p>
      <p>Suppose \(\{a_{n}\}\) is Cauchy. First it is bounded: taking \(\varepsilon = 1\), there is an \(N\) with \(\lvert a_{n} - a_{N} \rvert < 1\) for \(n \ge N\), hence \(\lvert a_{n} \rvert < \lvert a_{N} \rvert + 1\) for \(n \ge N\), and the finitely many earlier terms are bounded as well. By Bolzano–Weierstrass (Theorem 4.4) some subsequence converges, say \(a_{n_{j}} \to A\). We claim the whole sequence converges to \(A\). Given \(\varepsilon > 0\), choose \(N_{0}\) with \(\lvert a_{m} - a_{n} \rvert < \varepsilon/2\) for \(m, n \ge N_{0}\), and choose an index \(n_{j} \ge N_{0}\) with \(\lvert a_{n_{j}} - A \rvert < \varepsilon/2\). Then for every \(n \ge N_{0}\),</p>
      <p>\[ \lvert a_{n} - A \rvert \le \lvert a_{n} - a_{n_{j}} \rvert + \lvert a_{n_{j}} - A \rvert < \tfrac{\varepsilon}{2} + \tfrac{\varepsilon}{2} = \varepsilon. \]</p>
      <p>Hence \(a_{n} \to A\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <!-- expansion:caution — ROADMAP pitfall "Bolzano–Weierstrass dependency": make the logical chain back to completeness explicit -->
  <section class="env env-caution">
    <p class="env-head"><span class="env-kicker">Caution</span></p>
    <div class="env-body">
      <p>The Cauchy criterion is not an extra axiom; it is completeness in disguise. Trace the chain: Theorem 4.5 rests on Bolzano–Weierstrass (Theorem 4.4), which rests on the monotone-convergence property of \(\mathbb{R}\) (Theorem 4.1). Over the rationals every link breaks — there are bounded \(\mathbb{Q}\)-sequences that are Cauchy yet converge to no rational number (for instance, the partial sums of \(\sum 1/n!\), which approach the irrational \(e\)). So “Cauchy \(\Rightarrow\) convergent” is a statement about \(\mathbb{R}\) specifically, logically equivalent to its completeness.</p>
    </div>
  </section>

  <p>We can now discharge the deferred proof.</p>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span><span class="env-name">of Proposition 4.1</span></p>
    <div class="env-body">
      <p>Set \(S_{n} = \sum_{k=1}^{n} a_{k}\) and assume \(\sum_{k=1}^{\infty} \lvert a_{k} \rvert < \infty\). By Theorem 4.5 it suffices to show \(\{S_{n}\}\) is Cauchy. Given \(\varepsilon > 0\), convergence of the absolute series gives an \(N_{0}\) with \(\sum_{k = N_{0} + 1}^{\infty} \lvert a_{k} \rvert < \varepsilon\). Then for any \(m > n \ge N_{0}\), the triangle inequality yields</p>
      <p>\[ \lvert S_{m} - S_{n} \rvert = \Bigl\lvert \sum_{k = n + 1}^{m} a_{k} \Bigr\rvert \le \sum_{k = n + 1}^{m} \lvert a_{k} \rvert \le \sum_{k = N_{0} + 1}^{\infty} \lvert a_{k} \rvert < \varepsilon. \]</p>
      <p>So \(\{S_{n}\}\) is Cauchy, hence convergent, which is what it means for \(\sum_{k=1}^{\infty} a_{k}\) to converge. <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <h3 class="subsec-head">Extending \(e^{x}\) to all of \(\mathbb{R}\)</h3>

  <!-- expansion:intuition — use Proposition 4.1 to define e^x for x<0; record the absolute-value tail bound (**) that upgrades (*) to all real x -->
  <p>The convergence test does exactly the job we set it. For any real \(x\), the series of absolute values is \(\sum_{n=0}^{\infty} \lvert x \rvert^{n}/n! = e^{\lvert x \rvert}\), which is finite by Theorem 4.2 (or equals \(1\) when \(x = 0\)). Proposition 4.1 then guarantees that \(\sum_{n=0}^{\infty} x^{n}/n!\) itself converges. This defines</p>

  <!-- expansion:formula — the definition now covers all of ℝ, agreeing with Definition 4.1 for x>0 -->
  <p>\[ e^{x} = \sum_{n=0}^{\infty} \frac{x^{n}}{n!} \qquad \text{for every } x \in \mathbb{R}, \]</p>

  <p>in agreement with Definition 4.1 when \(x > 0\), and giving \(e^{0} = 1\). Moreover the tail bound upgrades to absolute values with no extra work: for \(\lvert x \rvert \le L\) and any \(k > n_{0} > 2L\), bounding each term of the remainder by its absolute value and reusing the geometric-tail estimate of \((*)\) (Strategy 4.1) gives</p>

  <!-- expansion:formula — absolute-value form of the §4.1 tail bound, valid for all |x| ≤ L -->
  <p>\[ \lvert e^{x} - P_{k}(x) \rvert \le \sum_{n = k + 1}^{\infty} \frac{\lvert x \rvert^{n}}{n!} \le \frac{L^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}}. \tag{$**$} \]</p>

  <p>The bound \((**)\) is the two-sided workhorse for the rest of the section: it is uniform over the whole symmetric interval \(\lvert x \rvert \le L\) and shrinks geometrically in \(k\). With it, the continuity proof for \(x > 0\) extends verbatim to every real base point.</p>

  <!-- expansion:intuition — Theorem 4.6 names full-line continuity (manuscript only states it in the Summary) -->
  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.6</span></p>
    <div class="env-body">
      <p>The function \(e^{x}\) is continuous on all of \(\mathbb{R}\).</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Fix any \(x_{0} \in \mathbb{R}\) and set \(L = \lvert x_{0} \rvert + 1\), so that every \(x\) with \(\lvert x - x_{0} \rvert < 1\) satisfies \(\lvert x \rvert \le L\). Repeat the argument of Theorem 4.3 word for word, with the one-sided bound \((*)\) replaced by its two-sided form \((**)\): given \(\varepsilon > 0\), choose \(k > n_{0} > 2L\) so that \(\dfrac{L^{n_{0}}}{n_{0}!}\bigl(\tfrac{1}{2}\bigr)^{k - n_{0}} < \varepsilon\), then use the continuity of the polynomial \(P_{k}\) to get \(\delta \in (0, 1)\) with \(\lvert P_{k}(x) - P_{k}(x_{0}) \rvert < \varepsilon\) whenever \(\lvert x - x_{0} \rvert < \delta\). The same three-term split gives \(\lvert e^{x} - e^{x_{0}} \rvert < 3\varepsilon\). Hence \(e^{x}\) is continuous at \(x_{0}\), and \(x_{0}\) was arbitrary. <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <h3 class="subsec-head">The exponent law</h3>

  <p>Everything is now in place for the multiplicative identity \(e^{x} e^{y} = e^{x + y}\). The proof is the continuity argument writ large: replace each exponential by its polynomial partial sum, recognize that the <em>product</em> of two partial sums reorganizes — through the binomial theorem — into the partial sum of \(e^{x+y}\) plus a controllable remainder, and let \((**)\) sweep the errors to zero.</p>

  <section class="env env-theorem">
    <p class="env-head"><span class="env-kicker">Theorem</span><span class="env-num">4.7</span><span class="env-name">Exponent law</span></p>
    <div class="env-body">
      <p>For all real numbers \(x\) and \(y\), \(\; e^{x} e^{y} = e^{x + y}\).</p>
    </div>
  </section>

  <!-- expansion:intuition — D3② (user-directed): full proof, not a 4-step outline. Six steps: two estimates, the product split, the binomial collapse of (I), the (II) tail bound, the four-term telescope, and the double limit. -->
  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>Fix \(x, y \in \mathbb{R}\) and choose \(L\) with \(\lvert x \rvert, \lvert y \rvert \le L\); throughout, take \(k > n_{0} > 8L\). Recall \(P_{k}(t) = \sum_{n=0}^{k} t^{n}/n!\).</p>

      <p class="para-head">Step 1: two estimates.</p>
      <p>From \((**)\), both \(\lvert e^{x} - P_{k}(x) \rvert\) and \(\lvert e^{y} - P_{k}(y) \rvert\) are at most \(\dfrac{L^{n_{0}}}{n_{0}!}\bigl(\tfrac{1}{2}\bigr)^{k - n_{0}}\). Also \(\lvert e^{y} \rvert \le \sum_{n=0}^{\infty} \lvert y \rvert^{n}/n! = e^{\lvert y \rvert} \le e^{L}\), and likewise \(\lvert P_{k}(x) \rvert \le e^{L}\).</p>

      <p class="para-head">Step 2: split the product.</p>
      <p>Multiplying the two finite sums and grouping the terms \(x^{i} y^{j}/(i!\,j!)\) by their total degree \(\ell = i + j\),</p>
      <p>\[ P_{k}(x)\, P_{k}(y) = \underbrace{\sum_{\ell = 0}^{k} \; \sum_{m = 0}^{\ell} \frac{x^{m} y^{\ell - m}}{m!\,(\ell - m)!}}_{\text{(I)}} \; + \; \underbrace{\sum_{\ell = k + 1}^{2k} \; \sum_{\substack{i + j = \ell \\ 0 \le i, j \le k}} \frac{x^{i} y^{j}}{i!\, j!}}_{\text{(II)}}. \]</p>
      <p>Group (I) collects every degree \(\ell \le k\), where no term is missing; group (II) collects the leftover high-degree terms \(k < \ell \le 2k\).</p>

      <p class="para-head">Step 3: the binomial theorem collapses (I).</p>
      <p>Writing \(\dfrac{1}{m!\,(\ell - m)!} = \dfrac{1}{\ell!}\dbinom{\ell}{m}\) and applying the binomial theorem \((x + y)^{\ell} = \sum_{m=0}^{\ell} \binom{\ell}{m} x^{m} y^{\ell - m}\) (with the manuscript’s notation \(\binom{\ell}{m} = C^{\ell}_{m}\)),</p>
      <p>\[ \text{(I)} = \sum_{\ell = 0}^{k} \frac{1}{\ell!} (x + y)^{\ell} = P_{k}(x + y). \]</p>

      <p class="para-head">Step 4: the leftover (II) is a tail.</p>
      <p>Enlarging the inner sum in (II) to all \(m\) with \(0 \le m \le \ell\) and passing to absolute values can only increase it, so</p>
      <p>\[ \lvert \text{(II)} \rvert \le \sum_{\ell = k + 1}^{2k} \frac{1}{\ell!} \sum_{m = 0}^{\ell} \binom{\ell}{m} \lvert x \rvert^{m} \lvert y \rvert^{\ell - m} = \sum_{\ell = k + 1}^{2k} \frac{(\lvert x \rvert + \lvert y \rvert)^{\ell}}{\ell!} \le \sum_{\ell = k + 1}^{\infty} \frac{(2L)^{\ell}}{\ell!}. \]</p>
      <p>Since \(\ell > n_{0} > 8L\), each ratio \((2L)/\ell < \tfrac{1}{4} < \tfrac{1}{2}\), so the same geometric estimate that produced \((*)\) gives</p>
      <p>\[ \lvert \text{(II)} \rvert \le \frac{(2L)^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}}, \qquad \text{hence} \qquad \bigl\lvert P_{k}(x) P_{k}(y) - P_{k}(x + y) \bigr\rvert \le \frac{(2L)^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}}. \]</p>

      <p class="para-head">Step 5: telescope the difference.</p>
      <p>Insert and remove the partial sums to compare \(e^{x} e^{y}\) with \(e^{x + y}\):</p>
      <p>\[ \begin{aligned} e^{x} e^{y} - e^{x + y} = {}&\bigl(e^{x} - P_{k}(x)\bigr) e^{y} + P_{k}(x)\bigl(e^{y} - P_{k}(y)\bigr) \\ &+ \bigl(P_{k}(x) P_{k}(y) - P_{k}(x + y)\bigr) + \bigl(P_{k}(x + y) - e^{x + y}\bigr). \end{aligned} \]</p>
      <p>Apply Step 1, Step 4, and the bound \((**)\) at \(x + y\) (valid since \(\lvert x + y \rvert \le 2L\), enlarging \(L\) to \(2L\) there). Each of the four pieces is at most a fixed constant times \(\bigl(\tfrac{1}{2}\bigr)^{k - n_{0}}\):</p>
      <p>\[ \lvert e^{x} e^{y} - e^{x + y} \rvert \le 2\, e^{L}\, \frac{L^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}} + 2\, \frac{(2L)^{n_{0}}}{n_{0}!} \left(\frac{1}{2}\right)^{k - n_{0}}. \]</p>

      <p class="para-head">Step 6: let \(k \to \infty\), then \(L \to \infty\).</p>
      <p>The right-hand side is independent of how the left-hand side was formed and tends to \(0\) as \(k \to \infty\), while the left-hand side does not depend on \(k\) at all. Therefore \(\lvert e^{x} e^{y} - e^{x + y} \rvert = 0\), i.e. \(e^{x} e^{y} = e^{x + y}\), for all \(x, y\) with \(\lvert x \rvert, \lvert y \rvert \le L\). Since \(L\) was arbitrary, the identity holds for all \(x, y \in \mathbb{R}\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <!-- expansion:intuition — Corollary 4.1 records positivity (manuscript's [e3] "0 < e^y" for y<0), now a one-line consequence of the exponent law -->
  <p>One loose end from the extension to negative arguments closes immediately.</p>

  <section class="env env-corollary">
    <p class="env-head"><span class="env-kicker">Corollary</span><span class="env-num">4.1</span></p>
    <div class="env-body">
      <p>\(e^{x} > 0\) for every real \(x\).</p>
    </div>
  </section>

  <section class="env env-proof">
    <p class="env-head"><span class="env-kicker">Proof</span></p>
    <div class="env-body">
      <p>By the exponent law, \(e^{x} = e^{x/2}\, e^{x/2} = \bigl(e^{x/2}\bigr)^{2} \ge 0\). And \(e^{x}\) is never zero, since \(e^{x} e^{-x} = e^{0} = 1\). Hence \(e^{x} > 0\). <span class="qed qed-proof"></span></p>
    </div>
  </section>

  <!-- expansion:summary — collect the section's results and hand off to §4.3 (the derivative) -->
  <p>Collecting the results of this section: the series</p>

  <p>\[ e^{x} = \sum_{n=0}^{\infty} \frac{x^{n}}{n!}, \qquad x \in \mathbb{R}, \]</p>

  <p>defines a function that is continuous on all of \(\mathbb{R}\) (Theorem 4.6), strictly positive (Corollary 4.1), and satisfies the exponent law \(e^{x} e^{y} = e^{x + y}\) (Theorem 4.7). The convergence that makes the definition legitimate for \(x < 0\) came from absolute convergence (Proposition 4.1), and that in turn from the Cauchy criterion (Theorem 4.5) and Bolzano–Weierstrass (Theorem 4.4) — the completeness of \(\mathbb{R}\) reappearing at every level. With \(e^{x}\) now established as a bona fide continuous, multiplicative, positive function, we are ready to return to the question that Chapter 2 answered only informally: its derivative. That rigorous re-derivation is the subject of §4.3, which leans on the same partial-sum machinery developed here.</p>
