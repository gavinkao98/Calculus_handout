L1-1：Def 2.3（differentiable）只有 corner 一種失敗模式
教學點：Definition 2.3 定義了 differentiable；散文描述了三種失敗模式（"no sharp corner, no vertical spike, and no break"），但 Example 2.11 只示範了 corner（\(|x|\)）。Vertical tangent（如 \(\sqrt[3]{x}\) at \(x=0\)）是不同類型的 non-differentiability（差商趨向 \(\pm\infty\) 而非左右不等），完全沒有 example 示範。
優先理由：教新動作——corner 是左右有限極限不等，vertical tangent 是差商無界；兩者的診斷步驟不同。

優先理由：函數類型單一（medium）。不過 §2.4 的 Example 2.14 已用 \(4e^x - x^3 + 2x\) 做了 higher derivatives（含指數），在一定程度上覆蓋了此缺口——因此降為選收。

§2.3 Differentiability, Continuity, and Higher Derivatives（補 2）

E2.3-a　Vertical tangent: \(f(x) = \sqrt[3]{x}\) at \(x = 0\)核心

來源 [source: CLP-1 §2.2, prob_s2.2 "f(x)=sqrt(1+x)" remark on vertical tangent + standard textbook example]（CC BY-NC-SA 4.0）・解材出處：authored（CLP1 有 \(\sqrt{1+x}\) 的 domain remark 但無 cube-root vertical tangent 的完整 worked example；solution 為本次撰寫）・插入：Example 2.11 之後、Figure 2.3 之前

填補 L1-1：Definition 2.3 的 informal gloss 提到「no vertical spike」，但 Example 2.11 只示範 corner。Vertical tangent 是差商趨向 \(\pm\infty\)（而非左右極限不等的 corner），診斷步驟不同：corner 看兩側有限極限是否相等；vertical tangent 看差商是否無界。這是一個新動作，需要獨立 example。

Show that \(f(x) = \sqrt[3]{x} = x^{1/3}\) is not differentiable at \(x = 0\), even though its graph has no corner there.

We form the difference quotient at \(a = 0\):

\[ \frac{f(0 + h) - f(0)}{h} \;=\; \frac{h^{1/3} - 0}{h} \;=\; \frac{1}{h^{2/3}}. \]

Since \(h^{2/3} > 0\) for all \(h \ne 0\), the quotient is always positive. As \(h \to 0\) the denominator \(h^{2/3} \to 0\), so

\[ \lim_{h \to 0} \frac{1}{h^{2/3}} = +\infty. \]

The limit does not exist as a finite number, so by Definition 2.3 the function is not differentiable at \(x = 0\).

Notice the contrast with Example 2.11. There the two one-sided limits existed but disagreed (\(+1\) vs \(-1\)); here both one-sided limits agree — they both tend to \(+\infty\) — so there is no corner. Instead the graph of \(\sqrt[3]{x}\) has a vertical tangent at the origin: the secant lines all steepen without bound as \(h \to 0\). A vertical line has no finite slope, and the derivative demands a finite limit, so differentiability fails for a different reason than at a corner.

O2.3-a　Piecewise differentiability: matching pieces選收

來源 [source: CLP-1 §2.2 prob_s2.2 "2006H piecewise x^2 / ax+b"]（CC BY-NC-SA 4.0）・解材出處：official（CLP1 附完整 solution；本書版為其改寫）・插入：Example 2.11 之後或 §2.3 末

填補 L2-1（選收）：把 differentiability 的定義「反過來用」——不是證明某函數不可微，而是找常數使 piecewise function 可微。這是 Theorem 2.1（differentiable \(\Rightarrow\) continuous）的直接應用：先用連續性得到一個方程，再用左右導數相等得到第二個方程。練習了新動作（反向使用定義）和結構對位（Theorem 2.1 的應用）。

Find the values of the constants \(a\) and \(b\) for which the function \[ f(x) = \begin{cases} x^{2} & x \le 2, \\ ax + b & x > 2 \end{cases} \] is differentiable everywhere.

When \(x \ne 2\) the function is a polynomial on each piece and therefore differentiable. The only point in question is \(x = 2\).

Step 1 — Continuity. By Theorem 2.1, differentiability at \(x = 2\) requires continuity there. Continuity forces the two pieces to meet: \[ \lim_{x \to 2^{-}} f(x) = 2^{2} = 4 \quad\text{and}\quad \lim_{x \to 2^{+}} f(x) = 2a + b, \] so we need \(2a + b = 4\).

Step 2 — Matching derivatives. The left-hand derivative at \(x = 2\) uses the \(x^{2}\) branch: its derivative is \(2x\), giving slope \(4\) at \(x = 2\). The right-hand derivative uses the \(ax + b\) branch: its derivative is the constant \(a\). For the two-sided limit to exist we need \[ a = 4. \]

From \(2a + b = 4\) with \(a = 4\) we get \(b = -4\). One can verify that with \(a = 4, b = -4\) the difference quotient at \(x = 2\) has a well-defined limit equal to \(4\).

改作差異：CLP1 原題的 solution 直接引用已知的 \(d/dx(x^2)=2x\)；本書在 §2.3 尚未正式介紹 differentiation rules（那是 §2.4），但 §2.2 Example 2.6 已從定義算出 \((x^2+4x-2)'=2x+4\)，故此處引用 §2.2 的結果是合理的。solution 措辭改為引用 Theorem 2.1 和 §2.2 的已知結果，而非 CLP1 的 Lemma 引用。

§2.4 Derivatives of Polynomials and the Exponential Function（補 2）

E2.4-a　Power rule for negative and fractional exponents核心

來源 [source: CLP-1 §2.6 prob_s2.6 "s(t)=3t^4+5t^3-1/t"]（CC BY-NC-SA 4.0）・解材出處：official（CLP1 附完整 solution；本書版為其改寫）・插入：Example 2.13 之後、exponential 小節之前

填補 L1-2：Caution 明確說 power rule 對 negative integer exponent 亦成立，但 Example 2.13 只用了正整數次方。此題把 \(1/t\) 改寫為 \(t^{-1}\) 再套 power rule——這正是 Caution 描述的動作，且驗證了 §2.2 Example 2.8 從 definition 得到的 \((1/x)'=-1/x^2\)。

Differentiate \(s(t) = 3t^{4} + 5t^{3} - \dfrac{1}{t}\).

To use the power rule on every term, we first rewrite \(1/t\) as a power: \(s(t) = 3t^{4} + 5t^{3} - t^{-1}\). Applying the power rule (including the negative-exponent case noted in the Caution above) to each term,

\[ \begin{aligned} s'(t) &= 3 \cdot 4t^{3} + 5 \cdot 3t^{2} - (-1)\,t^{-2} \\[2pt] &= 12t^{3} + 15t^{2} + \frac{1}{t^{2}}. \end{aligned} \]

The key step was rewriting \(1/t\) as \(t^{-1}\) so the power rule could act on it: the exponent drops by one (\(-1 \to -2\)) and the old exponent \(-1\) comes out front, producing the \(+1/t^{2}\) term — matching what we would obtain from the quotient rule (§2.5) or from the limit definition (§2.2 Example 2.8).

E2.4-b　Power rule with fractional exponent + tangent line核心

來源 [source: CLP-1 §2.4 prob_s2.4 "f(x)=3x^2+4x^(1/2)"]（CC BY-NC-SA 4.0）・解材出處：official（CLP1 附完整 solution；本書版為其改寫並加上 tangent-line 部分）・插入：E2.4-a 之後

填補 L1-2（延續）：E2.4-a 示範了 negative exponent；此題補上 fractional exponent \(x^{1/2}\)，並回扣 §2.1 的 tangent-line 計算——用 rule 而非 definition，示範「rules 確實讓生活更輕鬆」。驗證結果一致性：§2.2 Example 2.7 從 definition 得 \((x^{1/2})'=1/(2\sqrt{x})\)，此處 power rule 一行重現。

Differentiate \(f(x) = 3x^{2} + 4\sqrt{x}\) for \(x > 0\), and find the equation of the tangent line at \(x = 1\).

Writing \(\sqrt{x} = x^{1/2}\), we apply the power rule to each term:

\[ \begin{aligned} f'(x) &= 3 \cdot 2x + 4 \cdot \tfrac{1}{2}\,x^{-1/2} \\[2pt] &= 6x + \frac{2}{\sqrt{x}}. \end{aligned} \]

The term \(\frac{d}{dx}x^{1/2} = \tfrac{1}{2}x^{-1/2} = \tfrac{1}{2\sqrt{x}}\) is exactly the derivative we computed from the definition in §2.2 Example 2.7 — the power rule reproduces it in one step.

At \(x = 1\): \(f(1) = 3 + 4 = 7\) and \(f'(1) = 6 + 2 = 8\). The tangent line is \[ y - 7 = 8(x - 1), \quad\text{i.e.}\quad y = 8x - 1. \]

改作差異：CLP1 原題只要求 differentiate；本書版加上 tangent-line 計算以回扣 §2.1 的核心技能（Definition 2.1），使這道 example 同時服務兩個教學目的。原題的 \(f'(x) = 6x + 2/\sqrt{x}\) 不變。

§2.5 The Product and Quotient Rules（補 2）

E2.5-a　Quotient-rule error spotting核心

來源 [source: CLP-1 §2.6 prob_s2.6 Conceptual Q1 "Spot the error: f(x)=2x/(x+1)"]（CC BY-NC-SA 4.0）・解材出處：official（CLP1 附完整 solution；本書版為其改寫）・插入：Example 2.16 之後（quotient rule 第一個 full example 之後）

填補 L2-3：§2.5 的第三個 Caution 點名「swapping the order flips the sign」是最常見的 quotient-rule 錯誤。Example 2.16 的 solution 附帶一句提醒，但沒有 example 以辨錯為核心教學任務。此 CLP1 題正好是 error-spotting：展示一份含有 sign 和 algebra 錯誤的 quotient-rule 計算，要求學生找出並糾正。這是 Caution 的直接 worked example。

A student differentiates \(f(x) = \dfrac{2x}{x + 1}\) as follows:

\[ \begin{aligned} f'(x) &= \frac{2(x + 1) + 2x}{(x + 1)^{2}} \\[4pt] &= \frac{2(x + 1)}{(x + 1)^{2}} = \frac{2}{x + 1}. \end{aligned} \]

Spot all errors in this work, and give the correct derivative.

There are two errors.

Error 1 — Wrong sign in the quotient rule. The quotient rule gives \[ \left(\frac{f}{g}\right)' = \frac{f'g \;{\color{red}{-}}\; fg'}{g^{2}}. \] The student wrote \(+\) instead of \(-\) in the numerator. With top \(= 2x\) and bottom \(= x + 1\), the correct numerator is \[ (2)(x + 1) - (2x)(1) = 2x + 2 - 2x = 2, \] not \(2(x+1) + 2x = 4x + 2\).

Error 2 — Incorrect simplification. Even within the student's own (wrong) numerator, \(2(x+1) + 2x = 4x + 2\), not \(2(x+1)\). The student dropped the \(2x\) term.

The correct computation is \[ f'(x) = \frac{2(x + 1) - 2x}{(x + 1)^{2}} = \frac{2}{(x + 1)^{2}}. \] Notice the denominator is \((x+1)^2\), not \((x+1)\) — a third common slip that the student might have been heading toward.

O2.5-a　Quotient rule with exponential: \(e^x/(2x)\)選收

來源 [source: CLP-1 §2.7 prob_s2.7 Procedural "f(x)=e^x/(2x)"]（CC BY-NC-SA 4.0）・解材出處：official（CLP1 附完整 solution；本書版為其改寫）・插入：Example 2.17 之後（§2.5 末）

填補 L2-3（延續）：§2.5 目前的 quotient-rule examples 的分子/分母全是多項式（Ex 2.16: \(x/(x^2+1)\)）或 polynomial/\(e^x\)（Ex 2.17(c)）。此題把 \(e^x\) 放在分子、多項式放在分母——組合方向相反。且 solution 的 factoring out \(e^x\) 和最終化簡 \((x-1)e^x/(2x^2)\) 是一個新的代數動作。低優先，因 Ex 2.17(c) 已部分覆蓋。

Differentiate \(f(x) = \dfrac{e^{x}}{2x}\).

The quotient rule with top \(e^{x}\) and bottom \(2x\) gives

\[ \begin{aligned} f'(x) &= \frac{(e^{x})(2x) - (e^{x})(2)}{(2x)^{2}} \\[2pt] &= \frac{2e^{x}(x - 1)}{4x^{2}} = \frac{(x - 1)\,e^{x}}{2x^{2}}. \end{aligned} \]

Notice: the derivative of the top is again \(e^x\) (Theorem 2.5), and the derivative of the bottom is just \(2\). The sign of \(f'(x)\) changes at \(x = 1\) — for \(0 < x < 1\) we have \(f' < 0\) (the function is decreasing), and for \(x > 1\) we have \(f' > 0\) (the function is increasing).
