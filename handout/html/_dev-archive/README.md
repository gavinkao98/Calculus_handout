# _dev-archive —— 各章開發過程的歷史紀錄（凍結）

> **凍結存證，僅供回溯。** 本資料夾保留各章撰稿過程的開發產物——逐節的
> `seed_*`、`brief_*`、`PLAN-chNN.md`、`PROMPT-*-kickoff.md`、例題／圖稽核紀錄等，
> 是「當時各 session 實際拿到什麼」的存證。
>
> **路徑可能過時、不保證可用。** 這些檔案是按產出當下的工作目錄寫的，內含的相對路徑
> （例如 `exp-chNN/`、`new-chapter/`、`template-screen.html` 等）多為開發暫用或從未存在的位置，
> **搬遷後未逐條修正**——比照 [`../../../legacy/`](../../../legacy/) 的處置：當歷史紀錄保留、不竄改。
>
> **要看現行規範，不要看這裡：**
> - 撰稿流程（六階方向層）→ [`../../../CONTENT_DIRECTION.md`](../../../CONTENT_DIRECTION.md)
> - HTML 標記契約 → [`general/CONTRACT-html-writing.md`](general/CONTRACT-html-writing.md)
> - 排版指南 → [`../TYPESETTING_GUIDE.md`](../TYPESETTING_GUIDE.md)
> - 散文稽核 rubric → [`../_audit/PROSE-AUDIT-RUBRIC.md`](../_audit/PROSE-AUDIT-RUBRIC.md)
>
> 哪天要把某份 kickoff 當新章範本重用，**到時候只修那一份**的路徑即可。
