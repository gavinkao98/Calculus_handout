- **Load-bearing intuition (one):** slicing is a CHOICE of decomposition, and the right choice is the one whose slices see the region simply. Collision: washers about the \(y\)-axis for \(y=2x^2-x^3\) require inverting a cubic — the perpendicular slice fights the region. The shell = the OTHER natural slice: keep the vertical strip (which reads \(f\) directly) and let the rotation turn it into a nested cylinder — "tree rings" / "layers of an onion" instead of "coins in a stack".
- **Worked examples (motivating failure → between-curves → comparison):**

  - **Ex 7.10** — region between \(y = x\) and \(y = x^2\) about the \(y\)-axis: shell height = gap \(x - x^2\) (§7.1 echo), \(V = 2\pi\int_0^1 x(x - x^2)\,dx = \tfrac{\pi}{6}\).
