§4.4 Rolle’s Theorem and the Mean Value Theorem —— L2×2

L2：MVT／Rolle 作為「證明工具」從未示範。現有三例（Ex 4.2 解出 \(c=1\)、Ex 4.3/4.4 讀單調性）全部解出特定 \(c\) 或正面讀單調；沒有一例用 \(|f'|\) 的上界去 bound 函數值的差，也沒有一例把 Rolle 反過來用以排除第二根。

4.4-E1 highL2　MVT → Lipschitz 估計（只用存在性、不解 c）

Use the Mean Value Theorem to show that \(\lvert \sin a - \sin b \rvert \le \lvert a - b \rvert\) for all real numbers \(a\) and \(b\).

If \(a=b\) both sides are \(0\) and there is nothing to prove, so assume \(a \ne b\); relabel if necessary so that \(b < a\). The function \(f(x)=\sin x\) is continuous on \([b,a]\) and differentiable on \((b,a)\) (it is differentiable on all of \(\mathbb{R}\)), so by the Mean Value Theorem (Theorem 4.12) there is a point \(c \in (b,a)\) with \[ \frac{\sin a - \sin b}{a - b} = f'(c) = \cos c. \] We do not need to know which \(c\) — only that one exists. Multiplying across and taking absolute values, \[ \lvert \sin a - \sin b \rvert = \lvert \cos c \rvert \,\lvert a - b \rvert. \] Since \(\lvert \cos c \rvert \le 1\) for every real \(c\), this gives \(\lvert \sin a - \sin b \rvert \le \lvert a - b \rvert\), as claimed.

(The same argument bounds the change of any function by a bound on its derivative: if \(\lvert f' \rvert \le K\) throughout an interval, then \(\lvert f(a)-f(b) \rvert \le K\lvert a-b \rvert\) for any two points in it.)

why（teaching function）：教 MVT 的一個全新動作——把「導數有界」翻成「函數變化有界」（Lipschitz 估計）。緊接 Strategy 4.2 step 3「the theorem promises existence only … may rely on the existence of such a \(c\), but not on any specific value」的第一個正面落地：既有例都解出 \(c\)，本例刻意不解 \(c\)、只用 \(|\cos c|\le1\)。只用 Theorem 4.12 與 \(|\cos|\le1\)，高中自學可讀、無需新圖。

4.4-E2 highL2　Rolle 反用 → 精確根計數

How many real roots does \(f(x)=3x-\sin x\) have? Show that your count is exact.

By inspection \(f(0)=3\cdot 0-\sin 0 = 0\), so \(x=0\) is a root; \(f\) has at least one. The question is whether there could be a second. Suppose, for contradiction, that \(f\) had a second root \(a \ne 0\). Then \(f(0)=f(a)=0\), and \(f\) is continuous and differentiable on all of \(\mathbb{R}\), so Rolle's theorem (Theorem 4.11) applied to the interval between \(0\) and \(a\) would supply a point \(c\) strictly between them with \(f'(c)=0\). But \[ f'(x) = 3 - \cos x, \] and since \(\cos x \le 1 < 3\) for every real \(x\), we have \(f'(x) \ge 3-1 = 2 > 0\) everywhere — \(f'\) is never zero. No such \(c\) can exist, so the assumed second root is impossible.

Therefore \(f(x)=3x-\sin x\) has exactly one real root, namely \(x=0\). (The pattern is general: a function whose derivative never vanishes can have at most one root, because two roots would force a zero of \(f'\) between them.)
