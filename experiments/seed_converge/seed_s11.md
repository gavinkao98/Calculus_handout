# Seed — Section 1.1 Inverse Functions (sparse)

This is a SEED, not a finished section. It lists only the mathematical spine.
Expand it into a full handout section (the writing contract is in `rules.md`).

- A function is **one-to-one** when different inputs always give different outputs.
- **Horizontal line test**: a function is one-to-one if and only if no horizontal line meets its graph more than once.
- The **inverse function** `f^{-1}` is defined by: `f^{-1}(y) = x` exactly when `f(x) = y`. The domain and range swap.
- A function **has an inverse if and only if it is one-to-one**.
- **Composition identities**: `f^{-1}(f(x)) = x` and `f(f^{-1}(y)) = y`.
- **Reflection**: the graphs of `f` and `f^{-1}` are mirror images across the line `y = x`.
- **Procedure to find an inverse**: write `y = f(x)`; solve for `x` in terms of `y`; swap the names of `x` and `y`.
- Worked examples to include: `f(x) = x` (its own inverse); `g(x) = x^3` (inverse `∛x`); `h(x) = x^3 + 2` (inverse `∛(x − 2)`).
