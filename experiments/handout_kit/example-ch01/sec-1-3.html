1.3The Limit of a Function

A frequent setting in calculus is the need to study a ratio of changes. If \(s(t)\) is the position of a moving object at time \(t\), the ratio \[ \frac{s(t + h) - s(t)}{h} \] is the average velocity of the object from time \(t\) to time \(t + h\). Very often we want the instantaneous velocity — how fast the object is moving at a single moment — rather than the average. To describe this, we let \(h\) shrink toward zero and ask what value the ratio approaches. More generally, for any function \(f\), the ratio \[ \frac{f(y) - f(x)}{y - x} \] measures how \(f\) changes between two inputs. We may want to know what this ratio does as \(y\) gets closer and closer to \(x\), without ever equalling \(x\). This is the motivation for the concept of a limit.

Definition1.9

Suppose that \(f(x)\) is defined for all \(x\) near a number \(a\), except possibly at \(a\) itself. We write \[ \lim_{x \to a} f(x) = L \] and say that the limit of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) can be made arbitrarily close to \(L\) whenever \(x\) is sufficiently close to \(a\) but not equal to \(a\).

An alternative notation for the same idea is \[ f(x) \to L \qquad \text{as } x \to a, \] read “\(f(x)\) tends to \(L\) as \(x\) tends to \(a\).”

When we consider \(\lim_{x \to a} f(x)\), we are concerned with the values of \(f(x)\) for \(x\) near \(a\), not with the value of \(f(a)\). In particular, \(f(a)\) need not be defined at all. Figure 1.8 shows three functions whose values at \(a\) are different (and one is undefined), yet which all share the same limit as \(x \to a\).

Example1.15

The graph of a function \(f\) is shown in Figure 1.9. Evaluate the following, or state that the limit does not exist:

  1. \(\displaystyle\lim_{x \to -2} f(x)\).
  2. \(\displaystyle\lim_{x \to 0} f(x)\).
  3. \(\displaystyle\lim_{x \to 2} f(x)\).

Solution

  1. As \(x\) approaches \(-2\) from either side, the curve approaches height \(1\), so \(\displaystyle\lim_{x \to -2} f(x) = 1\).
  2. As \(x\) approaches \(0\), the curve approaches height \(0\), so \(\displaystyle\lim_{x \to 0} f(x) = 0\).
  3. As \(x\) approaches \(2\), the curve approaches height \(2\). The open dot at \((2, 2)\) and the solid dot at \((2, -2)\) tell us \(f(2) = -2\), but the limit ignores the value at the point: \(\displaystyle\lim_{x \to 2} f(x) = 2\), even though \(f(2) = -2\).

Example1.16

Sketch the graph of a function \(f\) with \(\displaystyle\lim_{x \to 3} f(x) = 10\) and \(f(3) = 0\).

Solution

Many answers are possible. Take any curve that approaches height \(10\) as \(x \to 3\) — for instance a straight line through \((3, 10)\) — and then move the single point at \(x = 3\) down to height \(0\), marking an open dot at \((3, 10)\) and a solid dot at \((3, 0)\). Near \(x = 3\), but not at it, the values approach \(10\), so \(\displaystyle\lim_{x \to 3} f(x) = 10\); the relocated point makes \(f(3) = 0\). The limit and the value at the point are independent, so nothing prevents this mismatch.

The limits above were guessed by looking at function values near the point of interest. In the following sections, we will develop systematic methods for evaluating limits and study more subtle behavior such as one-sided limits and infinite limits.
