1.1Inverse Functions

Recall that a function \(f\) assigns a value to every input in its domain. The reverse machinery — how to recover the input from the output — was named in the chapter overview; this section works it out. We proceed in three stages: identify which functions admit inversion (the one-to-one property), define the inverse function for those that do, and give an explicit procedure for constructing it. Before the formal definitions, two simple examples show when inversion is possible and when it is not.

  1. \(f(x) = x\) on \([0, 1]\). Here \(f(0) = 0\), \(f(\tfrac{1}{2}) = \tfrac{1}{2}\), \(f(\tfrac{1}{8}) = \tfrac{1}{8}\); each input produces its own distinct output.
  2. \(f(x) = x^{2}\) on \([-1, 1]\). Here \(f(0) = 0\), \(f(\tfrac{1}{2}) = \tfrac{1}{4}\), and \(f(-\tfrac{1}{2}) = \tfrac{1}{4}\); two different inputs produce the same output.

The difference between these two cases is the central point. In the first, every output corresponds to exactly one input. In the second, the output \(\tfrac{1}{4}\) corresponds to both \(\tfrac{1}{2}\) and \(-\tfrac{1}{2}\), so there is no unambiguous way to go backwards from output to input.

One-to-one functions

A function can be reversed only when each output comes from exactly one input. This motivates the following definition.

Definition1.1

Let \(f\) be a function with domain \(A\). We say that \(f\) is one-to-one if \[ f(x_{1}) \neq f(x_{2}) \qquad \text{whenever } x_{1} \neq x_{2}. \] Equivalently, \(f\) is one-to-one if \[ f(x_{1}) = f(x_{2}) \implies x_{1} = x_{2}. \]

Informally, a function is one-to-one when different inputs always produce different outputs.

Example1.1

A class of students is assigned identification numbers, arranged so that no two students share a number. Define \(f(\text{student})\) to be that student's identification number. Is \(f\) one-to-one?

Solution

By construction, no two students share a number, so distinct students always produce distinct identification numbers. Therefore \(f\) is one-to-one.

Now we return to the two mathematical examples from the opening.

Example1.2

Determine which of the following functions are one-to-one on their given domains: \[ \begin{aligned} f(x) &= x, & x &\in [0, 1],\\ g(x) &= x^{2}, & x &\in [-1, 1]. \end{aligned} \]

Solution

The function \(f(x) = x\) is one-to-one on \([0, 1]\) because each input is sent to itself, so distinct inputs always give distinct outputs.

The function \(g(x) = x^{2}\) is not one-to-one on \([-1, 1]\) because \[ \begin{aligned} g\!\left(\tfrac{1}{2}\right) &= \left(\tfrac{1}{2}\right)^{2} = \tfrac{1}{4},\\ g\!\left(-\tfrac{1}{2}\right) &= \left(-\tfrac{1}{2}\right)^{2} = \tfrac{1}{4}. \end{aligned} \] Two different inputs produce the same output, so \(g\) is not one-to-one.

The function \(g(x) = x^{2}\) on \([-1, 1]\) illustrates why not every function can be reversed: the value \(\tfrac{1}{4}\) is produced by both \(\tfrac{1}{2}\) and \(-\tfrac{1}{2}\), so no single rule can take \(\tfrac{1}{4}\) back to a uniquely determined input.

Looking at the graph of a function gives a quick way to decide whether it is one-to-one.

Remark1.1Horizontal line test

A horizontal line \(y = c\) meets the graph of \(f\) at a point \((x, c)\) exactly when \(f(x) = c\). Therefore, a function is one-to-one if and only if no horizontal line intersects its graph more than once. If the line \(y = c\) meets the graph at two distinct points \((x_{1}, c)\) and \((x_{2}, c)\), then \(f(x_{1}) = c = f(x_{2})\) with \(x_{1} \neq x_{2}\), so \(f\) fails to be one-to-one; conversely, if every horizontal line meets the graph at most once, each output corresponds to at most one input. Inverse functions

If \(f\) is one-to-one, then we can define a second function that undoes \(f\): given an output of \(f\), it returns the unique input that produced it.

Definition1.2

Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). The inverse function of \(f\), denoted by \(f^{-1}\), has domain \(B\) and range \(A\), and is defined by \[ f^{-1}(y) = x \quad \Longleftrightarrow \quad f(x) = y \ \text{ for any } y \in B. \]

Remark1.2

When we focus on the inverse function itself, it is often convenient to use \(x\) as its independent variable. In that case, we write \[ f^{-1}(x) = y \quad \Longleftrightarrow \quad f(y) = x. \]

Theorem1.1

A function has an inverse if and only if it is one-to-one.

Proof

Suppose first that \(f\) has an inverse \(f^{-1}\). If \(f(x_{1}) = f(x_{2})\), applying \(f^{-1}\) to both sides gives \[ f^{-1}\!\bigl(f(x_{1})\bigr) = f^{-1}\!\bigl(f(x_{2})\bigr), \] so \(x_{1} = x_{2}\). Therefore \(f\) is one-to-one. Conversely, suppose \(f\) is one-to-one with domain \(A\) and range \(B\). For each \(y \in B\) there exists some \(x \in A\) with \(f(x) = y\), and the one-to-one property forces this \(x\) to be unique. The correspondence \(y \mapsto x\) therefore defines an inverse function \(f^{-1} : B \to A\).

Example1.3

Find the inverse of each function.

  1. \(f(x) = x\).
  2. \(g(x) = x^{3}\).

Solution

  1. Since \(f(x) = x\) leaves every input unchanged, \(f\) is its own inverse, so \(f^{-1}(x) = x\).
  2. To find the inverse of \(g\), let \(y = x^{3}\). Solving for \(x\) gives \(x = \sqrt[3]{y}\). Interchanging the names of the variables, \(g^{-1}(x) = \sqrt[3]{x}\). As a check, \(g\!\bigl(g^{-1}(x)\bigr) = \bigl(\sqrt[3]{x}\bigr)^{3} = x\).

The defining relation \[ f^{-1}(y) = x \Longleftrightarrow f(x) = y \] yields two composition identities.

Proposition1.1

Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). Then \[ f^{-1}\!\bigl(f(x)\bigr) = x \ \text{ for every } x \in A, \qquad f\!\bigl(f^{-1}(y)\bigr) = y \ \text{ for every } y \in B. \]

Figure 1.2 illustrates both identities: \(f\) and \(f^{-1}\) travel between \(A\) and \(B\) in opposite directions, and composing them recovers the original input.

Graphically, if the graph of \(f^{-1}\) is drawn in the same coordinate plane as the graph of \(f\), the two graphs are reflections of one another across the line \(y = x\). The point \((a, b)\) lies on the graph of \(f\) exactly when \((b, a)\) lies on the graph of \(f^{-1}\).

Finding the inverse of a function

When \(f\) is one-to-one, we can usually construct its inverse by a short procedure.

Strategy1.1Finding the inverse of a one-to-one function

  1. Write \(y = f(x)\).
  2. Solve this equation for \(x\) in terms of \(y\).
  3. Interchange the names \(x\) and \(y\). The resulting equation gives \(y = f^{-1}(x)\).

Example1.4

Find the inverse of \(f(x) = x^{3} + 2\).

Solution

Let \(y = x^{3} + 2\). Solving for \(x\) gives \[ \begin{aligned} x^{3} &= y - 2,\\ x &= \sqrt[3]{y - 2}. \end{aligned} \] Interchanging the variables, \(y = \sqrt[3]{x - 2}\), so \(f^{-1}(x) = \sqrt[3]{x - 2}\). As a check, \[ f\!\bigl(f^{-1}(x)\bigr) = \left(\sqrt[3]{x - 2}\right)^{3} + 2 = (x - 2) + 2 = x. \]
