Summary

Key definitions.

  • One-to-one function — different inputs always produce different outputs.
  • Inverse function \(f^{-1}\) — reverses the action of a one-to-one function \(f\).
  • Inverse trigonometric functions \(\arcsin\), \(\arccos\), \(\arctan\), \(\arccsc\), \(\arcsec\), \(\arccot\) — defined by restricting each trigonometric function to a principal interval on which it is one-to-one.
  • Limit \(\lim_{x \to a} f(x) = L\) — informally, \(f(x)\) is arbitrarily close to \(L\) whenever \(x\) is sufficiently close to but not equal to \(a\); precisely, for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that \(|f(x) - L| < \varepsilon\) whenever \(0 < |x - a| < \delta\).
  • One-sided limits \(\lim_{x \to a^{-}} f(x)\) and \(\lim_{x \to a^{+}} f(x)\) — restrict the approach to values less than or greater than \(a\), respectively.
  • Infinite limit \(\lim_{x \to a} f(x) = \pm\infty\) — the values of \(f(x)\) grow without bound as \(x\) approaches \(a\).
  • Vertical asymptote at \(x = a\) — at least one of the one-sided limits at \(a\) is infinite.
  • Continuity at a point — \(\lim_{x \to a} f(x) = f(a)\); studied systematically in a later chapter.

Key theorems, propositions, and corollaries.

  • Horizontal line test — a function is one-to-one if and only if no horizontal line meets its graph more than once.
  • Existence of an inverse — a function has an inverse if and only if it is one-to-one.
  • Inverse trigonometric identities — on the principal intervals, \(\sin(\arcsin x) = x\) and \(\arcsin(\sin x) = x\), with analogous identities for \(\arccos\) and \(\arctan\).
  • Criterion for a two-sided limit — \(\lim_{x \to a} f(x) = L\) if and only if both one-sided limits at \(a\) equal \(L\).
  • Limit laws — limits respect sums, differences, products, quotients with nonzero denominator limit, and integer powers.
  • Squeeze theorem — if \(g(x) \le f(x) \le h(x)\) near \(a\) and \(g\) and \(h\) share a common limit \(L\) there, then so does \(f\).

Key formulas and identities.

  • Inverse composition: \(f^{-1}(f(x)) = x\) for \(x\) in the domain of \(f\), and \(f(f^{-1}(y)) = y\) for \(y\) in the range of \(f\).
  • Principal intervals of the inverse trigonometric functions: \(\arcsin x \in \left[-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right]\), \(\arccos x \in [0, \pi]\), \(\arctan x \in \left(-\tfrac{\pi}{2}, \tfrac{\pi}{2}\right)\).
  • Direct substitution: for polynomials and rational functions with nonzero denominator at \(a\), \(\lim_{x \to a} p(x) = p(a)\).
  • Two-sided criterion: \(\lim_{x \to a} f(x) = L\) exactly when \(\lim_{x \to a^{\pm}} f(x) = L\).
  • Precise \(\varepsilon\)-\(\delta\) statement: \(\lim_{x \to a} f(x) = L\) means that for every \(\varepsilon > 0\) there exists \(\delta > 0\) such that \(|f(x) - L| < \varepsilon\) whenever \(0 < |x - a| < \delta\).
