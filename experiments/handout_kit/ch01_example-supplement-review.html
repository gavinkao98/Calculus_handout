§1.1 Inverse Functions（補 3）

E1.1-a　代數判定一對一——直覺會失效的例子核心

來源 [source: Mooculus digInInversesOfFunctions L293]（CC BY-NC-SA 4.0——repo 授權 mooculus/license.md；該檔無逐檔 \license 覆寫）・插入：Example 1.2 之後

既有的 x 與 x² 都「看一眼就知道」；缺一個需要動手、且「三次方≈單調」直覺會翻車的例子。

Determine whether each function is one-to-one on \(\mathbb{R}\): \[ h(x) = x^{3} - 4x, \qquad k(x) = x^{3} + 4. \]

For \(h\), we look for two different inputs with the same output. Since \(h(x) = x(x^{2}-4) = x(x-2)(x+2)\), we have \(h(0) = h(2) = 0\) with \(0 \ne 2\), so \(h\) is not one-to-one — even though its formula looks similar to \(x^{3}\). For \(k\), suppose \(k(x_{1}) = k(x_{2})\). Then \(x_{1}^{3} + 4 = x_{2}^{3} + 4\), so \(x_{1}^{3} = x_{2}^{3}\), and taking cube roots gives \(x_{1} = x_{2}\). Therefore \(k\) is one-to-one. The lesson: two formulas of the same general shape can behave differently; when in doubt, test the definition directly.

E1.1-b　限制定義域修復不可逆函數（x² → √x）核心

來源 [source: Mooculus L407–453]・插入：y=x 鏡射段落之後・需新圖 1 幅

全節反覆說 x² 不可逆，但從未示範「怎麼修」。限制定義域是 §1.2 反三角函數的前置觀念，目前在 §1.2 開頭憑空出現——這個範例把兩節縫起來。

The function \(f(x) = x^{2}\) is not one-to-one on \(\mathbb{R}\). Restrict its domain so that the restricted function is one-to-one, and find the inverse of the restricted function.

On the interval \([0, \infty)\) the function \(f(x) = x^{2}\) is one-to-one: if \(x_{1}^{2} = x_{2}^{2}\) with \(x_{1}, x_{2} \ge 0\), then \(x_{1} = x_{2}\). To find the inverse, write \(x = \bigl(f^{-1}(x)\bigr)^{2}\); since \(f^{-1}(x) \ge 0\), taking the positive square root gives \[ f^{-1}(x) = \sqrt{x}, \qquad x \ge 0. \] This is exactly why \(\sqrt{x}\) means the positive square root: the convention makes the square root the inverse of the restricted squaring function. The figure shows the restricted parabola, its inverse, and the mirror line \(y = x\). The same idea — restrict first, then invert — is the key to the inverse trigonometric functions of the next section.

E1.1-c　有理函數求反＋雙向驗證核心

來源 [source: APEX §2.7 #7]（CC BY-NC 4.0；APEX 僅給題與指示，解為本次撰寫）・插入：Example 1.4 之後（節末）

Example 1.4 只驗證了單向；Proposition 1.1 的雙向恆等式沒有完整示範；Strategy 1.1 沒在非多項式上跑過。

Find the inverse of \(f(x) = \dfrac{3}{x-5}\) (for \(x \ne 5\)), and verify your answer by computing both \(f\bigl(f^{-1}(x)\bigr)\) and \(f^{-1}\bigl(f(x)\bigr)\).

Let \(y = \dfrac{3}{x-5}\). Solving for \(x\): \(x - 5 = \dfrac{3}{y}\), so \(x = 5 + \dfrac{3}{y}\). Interchanging the variables, \[ f^{-1}(x) = 5 + \frac{3}{x} = \frac{5x+3}{x}, \qquad x \ne 0. \] Verification, in both directions: \[ f\bigl(f^{-1}(x)\bigr) = \frac{3}{\bigl(5 + \tfrac{3}{x}\bigr) - 5} = \frac{3}{3/x} = x, \qquad f^{-1}\bigl(f(x)\bigr) = 5 + \frac{3}{3/(x-5)} = 5 + (x - 5) = x. \] Both compositions return the input, confirming Proposition 1.1. Note how the domains pair up: \(f\) excludes \(x = 5\) and \(f^{-1}\) excludes \(x = 0\), matching the fact that \(f\) never takes the value \(0\) and \(f^{-1}\) never takes the value \(5\).

O1.1-d　應用：攝氏 ↔ 華氏選收

來源 [source: Mooculus L181–227]・插入：§1.1 節末

The function \(f(t)=\tfrac95 t+32\) converts a temperature of \(t\) degrees Celsius into degrees Fahrenheit. What practical question does \(f^{-1}\) answer? Find a formula for \(f^{-1}(t)\), and verify your formula by computing \(f^{-1}(f(t))\).

\(f^{-1}\) converts Fahrenheit back to Celsius. Writing \(t = \tfrac95 f^{-1}(t) + 32\) and solving gives \(f^{-1}(t) = \tfrac59 (t-32)\); composing, \(f^{-1}(f(t)) = \tfrac59\bigl(\tfrac95 t + 32 - 32\bigr) = t\).

§1.2 Inverse Trigonometric Functions（補 3）

E1.2-a　arccos／arctan 的精確值（負輸入、鈍角主值、三角形法）核心

來源 [source: Stitz-Zeager via Mooculus 10_06_ex_131-154]（CC BY-NC-SA）・插入：Proposition 1.4 之後

數值範例只有 arcsin；arccos 的鈍角主值、arctan 的三角形、負輸入的符號處理全無示範。

Find the exact value of each expression.

(a) \(\sin\bigl(\arccos(-\tfrac{1}{2})\bigr)\)   (b) \(\sec(\arctan 10)\)   (c) \(\tan\bigl(\arcsin(-\tfrac{2\sqrt5}{5})\bigr)\)

(a) \(\arccos(-\tfrac12)\) is the angle in \([0,\pi]\) whose cosine is \(-\tfrac12\): that is \(\tfrac{2\pi}{3}\), and \(\sin\tfrac{2\pi}{3} = \tfrac{\sqrt3}{2}\). Note the principal value lands in the second quadrant — negative inputs to \(\arccos\) always do.

(b) Let \(\theta = \arctan 10\), so \(\tan\theta = 10\) with \(\theta \in (0, \tfrac{\pi}{2})\). In a right triangle with opposite side \(10\) and adjacent side \(1\), the hypotenuse is \(\sqrt{101}\), so \(\sec(\arctan 10) = \sqrt{101}\).

(c) Let \(\theta = \arcsin\bigl(-\tfrac{2\sqrt5}{5}\bigr)\), so \(\sin\theta = -\tfrac{2}{\sqrt5}\) with \(\theta \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}]\). Because \(\sin\theta \lt 0\), the angle lies in the fourth quadrant, where cosine is positive: \(\cos\theta = \sqrt{1-\tfrac45} = \tfrac{1}{\sqrt5}\). Therefore \(\tan\theta = \dfrac{-2/\sqrt5}{1/\sqrt5} = -2\). The sign came from the range of \(\arcsin\), not from guessing.

E1.2-b　複合函數的定義域核心

來源 [source: CLP-1 §2.12 #1 (a)(c)]（CC BY-NC-SA 4.0，官方詳解完整）・插入：Proposition 1.3 之後

本節建立了六張 domain／range 表，但沒有任何範例「使用」它們——「值域餵進定義域」的鏈式思考零示範。

Find the domain of each function.

(a) \(f(x) = \arcsin(\cos x)\)   (b) \(h(x) = \sin(\arccos x)\)

(a) Whatever \(x\) is, \(\cos x\) lies in \([-1,1]\), and \([-1,1]\) is exactly the domain of \(\arcsin\). So every real \(x\) is allowed: the domain of \(f\) is \(\mathbb{R}\).

(b) The inner function \(\arccos x\) requires \(x \in [-1,1]\); its output is an angle, and \(\sin\) accepts any angle. So the domain of \(h\) is \([-1,1]\). The pattern in both parts: the domain of a composition is governed by what the inner function accepts and what its outputs feed the outer function.

E1.2-c　其餘反三角函數的定義域＋本書分支約定的實際後果核心

來源 [source: CLP-1 §2.12 #5，依本書 arccsc 主值改作]・插入：Remark 1.4 之後（解答引用 Remark 1.4 的替代約定，須為回溯引用——Codex 稽核 advisory 修正，原定 Remark 1.3 之前）

「remaining functions」小節零範例；Remark 1.3/1.4 說分支約定會影響值，但沒有例子展示。全批改作幅度最大的一筆，請特別過目：CLP 原解用標準主值得 \(f(-1)=-\pi\)；本書的 arccsc 主值（Definition 1.6）給 \(\arccsc(-1)=\tfrac{3\pi}{2}\)，重算得 \(f(-1)=\pi\)，並把差異本身做成教學點。

Let \(f(x) = \arcsin x + \arccsc x\). Find the domain of \(f\), and evaluate \(f\) at every point of its domain.

\(\arcsin\) requires \(|x| \le 1\) while \(\arccsc\) requires \(|x| \ge 1\), so the domain is just the two points \(x = \pm 1\). At \(x = 1\): \(\arcsin 1 = \tfrac{\pi}{2}\) and \(\arccsc 1 = \tfrac{\pi}{2}\) (the angle in \(\bigl(0,\tfrac{\pi}{2}\bigr]\) with cosecant \(1\)), so \(f(1) = \pi\). At \(x = -1\): \(\arcsin(-1) = -\tfrac{\pi}{2}\), and under this text's principal range for \(\arccsc\) the angle with cosecant \(-1\) is \(\arccsc(-1) = \tfrac{3\pi}{2}\), so \(f(-1) = -\tfrac{\pi}{2} + \tfrac{3\pi}{2} = \pi\). So \(f\) is the constant \(\pi\) on its two-point domain. Under the alternative convention of Remark 1.4 (\(\arccsc(-1) = -\tfrac{\pi}{2}\)) one gets \(f(-1) = -\pi\) instead — a concrete instance of branch conventions changing answers, which is why this text fixes its conventions explicitly.

O1.2-d　主值的應用陷阱（粒子與 arccos(1)）選收

來源 [source: CLP-1 §2.12 #2]・插入：Proposition 1.3（arccos 恆等式）之後、E1.2-b 接續（題目用到 \(\arccos\)，不可早於 Definition 1.4——Codex 稽核 advisory 修正，原定「兩個 Caution 之後」過早）

A particle starts moving at time \(t=10\), and it bobs up and down so that its height at time \(t \ge 10\) is \(\cos t\). True or false: the particle has height \(1\) at time \(t = \arccos(1)\).

False. \(\cos t = 1\) for infinitely many \(t\), but \(\arccos\) returns only the principal value \(t = 0\) — and the particle is not even moving at \(t=0\) (its motion starts at \(t = 10\)). The particle has height \(1\) at the times \(2\pi n\) for integers \(n \ge 2\).

§1.3 The Limit of a Function（補 1）

E1.3-a　從圖讀極限（含 lim ≠ f(a) 的點）核心

來源 [source: CLP-1 §1.3 #1]（官方詳解完整）・插入：Figure 1.7 之後、Example 1.7 之前・需新圖 1 幅（連續波形，\((2,2)\) 空心、\((2,-2)\) 實心）

Figure 1.7 畫了「同極限、不同函數值」但沒有讀圖的 worked example；兩個既有範例都是列表法。

The graph of a function \(f\) is shown. Evaluate the following, or state that the limit does not exist:

(a) \(\lim_{x \to -2} f(x)\)   (b) \(\lim_{x \to 0} f(x)\)   (c) \(\lim_{x \to 2} f(x)\)

(a) As \(x\) approaches \(-2\) from either side, the curve approaches height \(1\), so the limit is \(1\). (b) As \(x\) approaches \(0\), the curve approaches height \(0\), so the limit is \(0\). (c) As \(x\) approaches \(2\), the curve approaches height \(2\) — the open dot at \((2,2)\) and the solid dot at \((2,-2)\) tell us \(f(2) = -2\), but the limit ignores the value at the point: the limit is \(2\), even though \(f(2) = -2\).

O1.3-b　構造：畫一條 lim=10 而 f(3)=0 的曲線選收

來源 [source: CLP-1 §1.3 #5]・插入：E1.3-a 之後・學生自己畫，無附圖成本

Draw the graph of a function \(f\) with \(\lim_{x \to 3} f(x) = 10\) and \(f(3) = 0\).

Many answers are possible: take any curve passing through height \(10\) at \(x = 3\), then move the single point at \(x=3\) down to height \(0\) (an open dot at \((3,10)\), a solid dot at \((3,0)\)). Near — but not at — \(x = 3\) the values approach \(10\), so the limit is \(10\), while \(f(3) = 0\).

§1.4 One-Sided and Infinite Limits（補 3）

E1.4-a　1/x 與 1/x²：DNE 和 ∞ 的分界核心

來源 [source: CLP-1 §1.3 #13–15]（官方詳解互相對照）・插入：Example 1.10 之後

手稿有 \(1/x^2 \to \infty\) 但沒有 \(1/x\)——「左右一正一負所以 DNE、且不寫成 ±∞」這個經典對照完全缺席。

Evaluate \(\lim_{x \to 0^{-}} \dfrac{1}{x}\), \(\lim_{x \to 0^{+}} \dfrac{1}{x}\), and \(\lim_{x \to 0} \dfrac{1}{x}\), and contrast the results with \(\lim_{x \to 0} \dfrac{1}{x^{2}}\).

As \(x\) approaches \(0\) from the left, \(\tfrac1x\) is negative with larger and larger magnitude, so \(\lim_{x \to 0^{-}} \tfrac1x = -\infty\). From the right, \(\tfrac1x\) is positive and grows without bound, so \(\lim_{x \to 0^{+}} \tfrac1x = \infty\). Because the two one-sided behaviors disagree, the two-sided limit does not exist — and since the values do not all grow in the same signed direction, we do not write \(\lim_{x \to 0} \tfrac1x = \infty\). Contrast \(\tfrac{1}{x^{2}}\): both sides grow positively, so there we do write \(\lim_{x \to 0} \tfrac{1}{x^{2}} = \infty\).

E1.4-b　由部分資訊推論：判準的雙向使用核心

來源 [source: CLP-1 §1.3 #8, #9]（官方詳解完整）・插入：Proposition 1.5 之後、Example 1.9 之前

兩側判準只在「算出兩側→判定雙側」方向用過；「已知雙側→推單側」與「只知單側→資訊不足」從未演示。

Suppose \(f\) is defined on all of \(\mathbb{R}\).

(a) If \(\lim_{x \to -2} f(x) = 16\), what is \(\lim_{x \to -2^{-}} f(x)\)?
(b) If instead all we know is \(\lim_{x \to -2^{-}} f(x) = 16\), what can we say about \(\lim_{x \to -2} f(x)\)?

(a) For the two-sided limit to equal \(16\), both one-sided limits must equal \(16\); in particular \(\lim_{x \to -2^{-}} f(x) = 16\).

(b) Not enough information. If the right-hand limit also equals \(16\), then \(\lim_{x \to -2} f(x) = 16\); if it differs (or fails to exist), the two-sided limit does not exist. The criterion transfers information from the two-sided limit to each side, but only both sides together determine the two-sided limit.

E1.4-c　分母為零不保證鉛直漸近線核心

來源 [source: CLP-1 §3.6.1 #1]（官方詳解完整）・插入：Example 1.11／Figure 1.10 之後

Example 1.11 只示範「候選點成功變成漸近線」；「候選點失敗」（洞）從未出現，而這是最常見的誤判。

Suppose \(f(x) = \dfrac{g(x)}{x^{2}-9}\) for some function \(g\). Must the line \(x = -3\) be a vertical asymptote of \(y = f(x)\)?

No. The denominator does vanish at \(x = -3\), so \(x = -3\) is a candidate — but whether it is an asymptote depends on the numerator. If \(g(x) = 1\), then \(f(x) = \tfrac{1}{x^{2}-9}\) blows up near \(-3\), and \(x = -3\) is a vertical asymptote. But if \(g(x) = x^{2}-9\), then \(f(x) = 1\) at every point of its domain, and there is no asymptote at all. A zero denominator marks a place to investigate with one-sided limits, not a guaranteed asymptote.

§1.5 Limit Laws and Computational Techniques（補 3）

E1.5-a　抽象地使用 limit laws核心

來源 [source: CLP-1 §1.4 #42]（官方解完整）・插入：Theorem 1.2 口頭重述段之後、「Direct substitution」之前

Theorem 1.2 之後直接跳到具體多項式；「不知道 \(f\) 是什麼、只知道它的極限」的抽象使用零示範——而這正是 laws 的本義。

Suppose \(\lim_{x \to -1} f(x) = -1\). Evaluate \[ \lim_{x \to -1} \frac{x\,f(x) + 3}{2 f(x) + 1}. \]

We never need a formula for \(f\). By the product law, \(\lim_{x \to -1} x f(x) = (-1)(-1) = 1\); by the sum and constant-multiple laws the numerator tends to \(1 + 3 = 4\) and the denominator tends to \(2(-1) + 1 = -1\). The denominator's limit is nonzero, so the quotient law applies: \[ \lim_{x \to -1} \frac{x f(x) + 3}{2 f(x) + 1} = \frac{4}{-1} = -4. \]

E1.5-b　0/0 可以是任何值核心

來源 [source: CLP-1 §1.4 #2, #5]（官方解完整）・插入：「Algebraic simplification」首段之後、Example 1.14 之前

「indeterminate」一詞出現了，但「為什麼」不定（可以是任何值）沒有示範；學生普遍誤信 0/0 必為 1 或 0。

(a) Find functions \(f\) and \(g\) with \(\lim_{x \to 3} f(x) = \lim_{x \to 3} g(x) = 0\) and \(\lim_{x \to 3} \dfrac{f(x)}{g(x)} = 10\).
(b) What are the possible values of \(\lim_{x \to 3} \dfrac{f(x)}{g(x)}\), given only that both \(f\) and \(g\) tend to \(0\)?

(a) Take \(f(x) = 10(x-3)\) and \(g(x) = x - 3\). Both tend to \(0\) at \(3\), yet \(\dfrac{f(x)}{g(x)} = 10\) wherever it is defined, so the quotient's limit is \(10\).

(b) Replacing \(10\) by any constant \(c\) gives limit \(c\); choices such as \(f(x) = x-3\), \(g(x) = (x-3)^{3}\) give \(\infty\); swapping signs gives \(-\infty\); and \(f(x) = x-3\), \(g(x) = |x-3|\) makes the limit fail to exist. A \(\tfrac00\) form can therefore equal anything — which is exactly why it is called indeterminate, and why we must simplify before judging.

E1.5-c　反推常數使極限存在（壓軸綜合）核心

來源 [source: CLP-1 §1.4 #43]（官方解完整）・插入：Strategy 1.2 之後（節末）

所有既有範例都「正向計算」；缺一個先推理（分母趨 0 ⟹ 分子必須趨 0）再計算的綜合示範，收束全節。

Find the value of the constant \(a\) for which \(\lim_{x \to -2} \dfrac{x^{2} + a x + 3}{x^{2} + x - 2}\) exists, and evaluate the limit for that value of \(a\).

The denominator factors as \((x+2)(x-1)\) and tends to \(0\) as \(x \to -2\). If the numerator tended to a nonzero number, the quotient would blow up and the limit could not exist; so the numerator must also tend to \(0\): \(4 - 2a + 3 = 0\), giving \(a = \tfrac{7}{2}\). With this \(a\), \[ \lim_{x \to -2} \frac{x^{2} + \tfrac{7}{2}x + 3}{x^{2} + x - 2} = \lim_{x \to -2} \frac{(x+2)\bigl(x + \tfrac{3}{2}\bigr)}{(x+2)(x-1)} = \lim_{x \to -2} \frac{x + \tfrac{3}{2}}{x - 1} = \frac{-\tfrac12}{-3} = \frac{1}{6}. \] Notice the flow of reasoning: the existence of the limit forced the value of \(a\), and then the techniques of this section finished the computation.

O1.5-d　通分化簡選收

來源 [source: CLP-1 §1.4 extra（prob_s1.4.tex:751）]（官方解完整）・插入：Example 1.15 之後

Strategy 1.2 第 2 步點名了「combine fractions」但沒有範例做過。代數較重，由你定奪。

Evaluate \(\displaystyle\lim_{t \to \frac{1}{2}} \frac{\frac{1}{3t^{2}}+\frac{1}{t^{2}-1}}{2t-1}\).

Combining the fractions over the common denominator \(3t^{2}(t^{2}-1)\) turns the numerator into \(\dfrac{4t^{2}-1}{3t^{2}(t^{2}-1)} = \dfrac{(2t+1)(2t-1)}{3t^{2}(t^{2}-1)}\); the factor \(2t-1\) cancels with the denominator, leaving \(\lim_{t \to 1/2} \dfrac{2t+1}{3t^{2}(t^{2}-1)} = \dfrac{2}{\tfrac34 \cdot (-\tfrac34)} = -\dfrac{32}{9}\).

§1.6 The Precise Definition of a Limit（補 2）

E1.6-a　診斷一個寫錯的定義核心

來源 [source: APEX §1.2 #1, #2]（官方解完整）・插入：Definition 1.13 解說段之後、uniqueness 之前

定義的「量詞順序與蘊含方向」只靠 prose 講；沒有讓學生主動辨錯的示範——這是 ε-δ 最高頻的誤解。

What is wrong with the following attempted definition of \(\lim_{x \to a} f(x) = L\)?

“Given any \(\delta \gt 0\), there exists \(\varepsilon \gt 0\) such that whenever \(|f(x) - L| \lt \varepsilon\), we have \(0 \lt |x - a| \lt \delta\).”

Two things are reversed. First, the order of the tolerances: the challenge must come on the \(y\)-side — “given any \(\varepsilon\)” — and \(\delta\) is our response; the attempted version lets us pick the easy quantity after seeing the hard one. Second, the direction of the implication: the definition must say that closeness of \(x\) to \(a\) forces closeness of \(f(x)\) to \(L\), i.e. \(0 \lt |x-a| \lt \delta\) implies \(|f(x) - L| \lt \varepsilon\) — not the other way around. Keeping both corrections gives exactly Definition 1.13.

E1.6-b　用已知不等式餵 ε-δ：lim sin x = 0核心

來源 [source: APEX §1.2 #8]（官方解完整）・插入：Example 1.22 之後、Definition 1.14 之前

既有兩個證明都是「代數整理型」；「借一個已知不等式直接餵進定義」是第三種典型套路，並為 squeeze 式思維鋪路。

Using the fact that \(|\sin x| \le |x|\) for all \(x\), show from the definition that \(\lim_{x \to 0} \sin x = 0\).

Given \(\varepsilon \gt 0\), choose \(\delta = \varepsilon\). If \(0 \lt |x - 0| \lt \delta\), then \[ |\sin x - 0| = |\sin x| \le |x| \lt \delta = \varepsilon. \] Therefore \(\lim_{x \to 0} \sin x = 0\). The borrowed inequality did all the work: no factoring, no auxiliary bound — Step 2 of Strategy 1.3 was handed to us for free.

O1.6-c　常數函數：任何 δ 都行選收

來源 [source: APEX §1.2 #3]・插入：Example 1.21 之前

「δ 可以隨便選」是個小而亮的觀念點（學生常以為 δ 必須是 ε 的某個函數）。

Show from the definition that \(\lim_{x \to 2} 5 = 5\).

Given \(\varepsilon \gt 0\), we need \(|5 - 5| \lt \varepsilon\) whenever \(0 \lt |x-2| \lt \delta\) — but \(|5-5| = 0 \lt \varepsilon\) is always true, no matter what \(x\) does. So any choice works; we may simply take \(\delta = \varepsilon\) (or \(\delta = 1\), or anything else).

O1.6-d　Definition 1.14 的 M-δ 示範AI 出題選收

來源 [source: AI]（題庫無乾淨候選）・插入：Definition 1.14 解說段之後

無限極限的精確定義目前零示範。收＝授權我出以下這題、你再審。

Show from the precise definition that \(\lim_{x \to 0} \dfrac{1}{x^{2}} = \infty\).

Given \(M \gt 0\), choose \(\delta = \dfrac{1}{\sqrt{M}}\). If \(0 \lt |x| \lt \delta\), then \(x^{2} \lt \delta^{2} = \dfrac{1}{M}\), so \(\dfrac{1}{x^{2}} \gt M\). Since every threshold \(M\) is eventually exceeded, \(\lim_{x \to 0} \tfrac{1}{x^{2}} = \infty\) — the rigorous counterpart of Example 1.10.
